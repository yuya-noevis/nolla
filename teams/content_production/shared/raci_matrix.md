# RACI マトリクス

各工程における責任分担。1工程のAccountable（A）は必ず1名。

R = Responsible（実行）/ A = Accountable（承認権限）/ C = Consulted（相談）/ I = Informed（報告先）

## 読み方

- **R（実行）**: 実際に手を動かす人
- **A（承認）**: 「これでOK」を出す権限を持つ人。1工程に1人だけ
- **C（相談）**: 専門知識を提供し、フィードバックする人。双方向のやり取りが発生
- **I（報告）**: 結果を共有される人。意思決定には参加しない

### 重要な原則
- Aが承認しない限り、次のStepに進まない
- CとAが意見で対立した場合、Aが最終判断を行う
- ただし **療育的正確性に関する判断は、常に療育設計エージェント（02）が最終権限** を持つ（プロデューサーであっても科学的判断を覆さない）
