# 品質チェックリスト

全動画の公開前に必ず確認する。チェックが1つでも不合格の場合、該当エージェントに差し戻す。

---

## 1. 療育的正確性（療育設計エージェントが確認）

- [ ] 療育目標シートに記載したターゲット行動が動画に反映されている
- [ ] 使用語彙がレベル別リストの範囲内である
- [ ] インバーバル・フィルインの「間」が指定秒数（±0.5秒）で配置されている
- [ ] プロンプトの種類（音声/視覚/ジェスチャー）が療育目標シート通りである
- [ ] ソーシャルストーリーの場合、文型比率（記述文:コーチング文 ≥ 2:1）を遵守している
- [ ] 受動視聴にならない能動的参加ポイントが最低1回/30秒ある
- [ ] 不適切な行動の強化につながる表現がない
- [ ] エビデンス根拠が明記可能（どの手法に基づいているか説明できる）

## 2. 感覚配慮（ビジュアル + サウンドエージェントが確認）

### 視覚
- [ ] フラッシュ・点滅効果がない（1秒間に3回以上の明滅がゼロ）
- [ ] カット間隔が2秒以上（推奨4-6秒）
- [ ] ネオンカラー・蛍光色の大面積使用がない
- [ ] 高コントラストの縞模様がない
- [ ] 画面上の同時情報が3要素以下（キャラ+背景+1テキスト程度）
- [ ] Nollaカラーパレットに準拠している

### 聴覚
- [ ] 急激な音量変化がない（10dB以上の急上昇がゼロ）
- [ ] BGMとナレーションの音量比が適切（BGM ≤ ナレーションの60%）
- [ ] 甲高い効果音の連続がない
- [ ] 予告なしの大きな音がない
- [ ] BGMのBPMが90-110の範囲内

## 3. コンテンツ構成（脚本エージェントが確認）

- [ ] 構成テンプレート（30秒 or 60秒）に準拠している
- [ ] オープニングが統一フォーマットである
- [ ] エンディングが統一フォーマットである
- [ ] 1動画のテーマが1つに絞られている
- [ ] 最初の2-3秒にビジュアルフックがある
- [ ] ループ構造が設計されている（終わりが始まりにつながる）

## 4. ビジュアル品質（ビジュアルエージェントが確認）

- [ ] キャラクターがスタイルガイドに準拠している（顔・色・体型の一貫性）
- [ ] 背景がシンプルで注目対象が明確
- [ ] レベル別サムネイル色が正しい
- [ ] 色覚多様性への配慮（赤緑の直接隣接がない、色+記号のダブルコーディング）
- [ ] テロップが読みやすい（サイズ・コントラスト・表示時間）

## 5. サウンド品質（サウンドエージェントが確認）

- [ ] ナレーションが明瞭で聞き取りやすい
- [ ] 効果音が動きと同期している
- [ ] 「間」の区間で無音 or 極小BGMのみになっている
- [ ] 全体の音量がプラットフォーム推奨レベル（-14 LUFS前後）

## 6. 技術品質（動画制作エージェントが確認）

- [ ] 解像度が1080x1920（縦型9:16）
- [ ] フレームレートが30fps以上
- [ ] 映像にアーティファクト（AI生成の乱れ）がない
- [ ] 音声と映像が同期している
- [ ] ファイルサイズがプラットフォーム上限内
- [ ] サムネイルが作成されている

## 7. コンプライアンス（プロデューサーが確認）

- [ ] 子どもの安全を脅かす表現がない
- [ ] 差別的・排他的な表現がない
- [ ] 広告的な表現がない（特定商品の宣伝等）
- [ ] COPPA準拠（個人情報収集を促す要素がない）
- [ ] YouTube Kids / TikTokの子ども向けコンテンツポリシーに準拠
- [ ] 著作権侵害がない（音楽・キャラクター・映像素材）

---

## 判定基準

| 判定 | 条件 |
|------|------|
| **Go（公開可）** | 全チェック項目がパス |
| **Rework（差し戻し）** | 1項目でも不合格 → 該当エージェントに差し戻し |
| **Hold（保留）** | 判断に迷う項目がある → プロデューサー + 療育設計者で協議 |
