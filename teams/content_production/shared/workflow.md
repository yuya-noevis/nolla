# 制作ワークフロー

## 1本の動画制作フロー（6ステップ）

Sesame Workshop CTWモデルを簡略化。「共同設計モデル」を採用し、療育設計者が台本の初期段階から関与する。

---

### Step 1: 療育目標設定（療育設計 → プロデューサー承認）

**担当**: 療育設計エージェント
**成果物**: 療育目標シート

| 項目 | 内容 |
|------|------|
| 対象領域 | 発語 / コミュニケーション / 社会性 / 読み書き / ライフスキル |
| 療育アプローチ | TOP5から選択（ソーシャルストーリー / ビデオモデリング+フィルイン / 歌リズム / フォニックス / ライフスキル） |
| ターゲット行動 | 「動画視聴後に子どもが〇〇する」の具体的記述 |
| 使用語彙リスト | レベル別の理解語彙・表出語彙 |
| インバーバル設計 | 「間」の位置・秒数・プロンプトの種類 |
| レベル | 入門 / 初級 / 中級 / 上級 / スペシャル |
| エビデンス根拠 | 参照する研究・手法名 |

**完了条件**: プロデューサーが療育目標シートを承認

---

### Step 2: 脚本・構成（脚本エージェント → 療育設計レビュー）

**担当**: 脚本・構成エージェント
**成果物**: 台本（タイムライン付き）

| 項目 | 内容 |
|------|------|
| 構成テンプレート | 30秒版 or 60秒版を選択 |
| シーン記述 | 各シーンの映像・音声・テキストを秒単位で記述 |
| セリフ・ナレーション | 療育目標シートの語彙リストに準拠 |
| 「間」の配置 | インバーバル・フィルインの正確な位置と秒数 |
| 参加ポイント | 子どもが声を出す/動くタイミング |

**レビューフロー**:
1. 脚本エージェントが下書き作成
2. 療育設計エージェントが科学的正確性をレビュー（「間」の秒数、プロンプト配置、語彙の適切性）
3. 修正→再レビュー→確定

**完了条件**: 療育設計エージェントが台本を承認

---

### Step 3: ビジュアルデザイン（ビジュアルエージェント）

**担当**: ビジュアルデザインエージェント
**成果物**: ストーリーボード（絵コンテ）+ スタイルガイド（初回のみ）

| 項目 | 内容 |
|------|------|
| キャラクター | スタイルガイドに準拠したキャラクター配置 |
| 背景 | シーンごとの背景設計 |
| カラー | Nollaカラーパレットに準拠 |
| レベル色 | サムネイルの背景色をレベルに合わせて設定 |
| 感覚配慮 | 避けるべき要素リストとの照合 |

**完了条件**: プロデューサーがストーリーボードを承認

---

### Step 4: サウンドデザイン（サウンドエージェント）

**担当**: サウンドデザインエージェント
**成果物**: 音声設計書 + BGM/SE素材 + ナレーション

| 項目 | 内容 |
|------|------|
| BGM | BPM・キー・楽器構成の指定 |
| 効果音 | 各アクションに紐づくSEリスト |
| ナレーション | 速度・ピッチ・トーンの指定 + 収録/AI生成 |
| 音量バランス | BGM:ナレーション = 50-60%:100% |
| 感覚配慮 | 音圧上限・急激な変化の排除 |

**完了条件**: 療育設計エージェントが音の感覚配慮を確認

---

### Step 5: 動画制作・編集（動画制作エージェント）

**担当**: 動画制作・編集エージェント
**成果物**: 完成動画

| 項目 | 内容 |
|------|------|
| AI動画生成 | Kling AI等でアニメーション生成 |
| キャラ一貫性 | スタイルガイドとの照合 |
| 編集 | CapCutでタイミング調整・テロップ・SE合成 |
| テンポチェック | カット間隔4-6秒、BPM 90-110の範囲内か |

**完了条件**: 技術的な品質基準をクリア

---

### Step 6: 最終チェック・公開（プロデューサー）

**担当**: プロデューサー
**成果物**: 公開判断（Go / Rework）

| チェック項目 | 内容 |
|---|---|
| 療育目標との整合 | 目標シートの意図が動画に反映されているか |
| 品質チェックリスト | `quality_checklist.md` の全項目をパス |
| ブランド一貫性 | トーン・キャラ・色がNollaの世界観に合っているか |
| プラットフォーム最適化 | サムネイル・タイトル・ハッシュタグ・説明文 |

**完了条件**: プロデューサーがGoを出す → 公開
