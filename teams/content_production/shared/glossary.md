# 用語集

Nollaの動画制作で使う主要な専門用語。全エージェントが共通の認識で使用する。

---

## 療育・発達支援

| 用語 | 読み | 説明 |
|------|------|------|
| ASD | エーエスディー | 自閉スペクトラム症。社会的コミュニケーションと行動パターンに特性がある |
| ADHD | エーディーエイチディー | 注意欠如・多動症。注意の維持、衝動性、多動に特性がある |
| LD | エルディー | 学習障害。読み書き・計算など特定の学習に困難がある |
| ST | エスティー | 言語聴覚士（Speech-Language-Hearing Therapist） |
| SLP | エスエルピー | 言語療法士（Speech-Language Pathologist）。STの英語圏の呼称 |
| OT | オーティー | 作業療法士（Occupational Therapist） |
| ABA | エービーエー | 応用行動分析（Applied Behavior Analysis）。行動の原理に基づく介入手法 |
| BCBA | ビーシービーエー | 認定行動分析士（Board Certified Behavior Analyst） |
| ビデオモデリング | — | 適切な行動の手本を映像で見せて模倣を促す手法 |
| ソーシャルストーリー | — | 社会的場面を短い物語で説明し適切な行動を教える手法（Carol Gray考案） |
| インバーバル・フィルイン | — | 歌やフレーズの一部を空けて子どもに言わせる手法。例：「きらきら ひかる おそらの…」→子「ほし」 |
| パレンティーズ | — | 高いピッチ・ゆっくり・誇張した抑揚で話す話し方。乳幼児の言語発達を促進する |
| ビデオプロンプティング | — | 課題の手順を動画で段階的に示し、自立を促す手法 |
| フォニックス | — | 文字と音の対応関係を体系的に教える読み書き指導法 |
| 感覚統合 | かんかくとうごう | 視覚・聴覚・触覚等の感覚情報を脳が適切に処理・統合する機能 |
| 感覚過敏 | かんかくかびん | 特定の感覚刺激に対して通常より強く反応する特性 |
| 般化 | はんか | 学んだスキルを別の場面・相手・環境でも使えるようになること |
| プロンプト | — | 子どもが正しい行動をするための手がかり（音声・視覚・身体的等） |
| フェイディング | — | プロンプトを段階的に減らし、自立を促すこと |
| 強化 | きょうか | 行動の直後に好ましい結果を与え、行動の頻度を増やすこと |
| エコラリア | — | 聞いた言葉やフレーズをそのまま繰り返す特性。学習メカニズムとしても機能する |
| グレーゾーン | — | 診断基準を完全には満たさないが、発達に課題がある状態 |

## コンテンツ制作

| 用語 | 説明 |
|------|------|
| フック | 動画の最初の2-3秒で視聴者の注意を引く要素 |
| ループ構造 | 動画の終わりが始まりにつながり、繰り返し再生を促す設計 |
| 完了率（完視聴率） | 動画を最後まで見た視聴者の割合 |
| カット間隔 | 映像の切り替わりの時間間隔。Nolla推奨: 4-6秒 |
| BPM | Beats Per Minute。音楽のテンポ。Nolla推奨: 90-110 |
| LUFS | Loudness Units relative to Full Scale。音量の統一基準。推奨: -14 LUFS |
| トンマナ | トーン＆マナー。コンテンツ全体の雰囲気・スタイルの一貫性 |
| スタイルガイド | キャラクター・色・フォント等のデザインルールを定めた文書 |
| ストーリーボード | 動画の各シーンをコマ割りで示した設計図（絵コンテ） |
| サムネイル | 動画一覧で表示される代表画像 |

## AI動画生成

| 用語 | 説明 |
|------|------|
| プロンプト | AI生成ツールに与える指示文 |
| リファレンス画像 | AI生成の出力スタイルを指定するための参考画像 |
| アーティファクト | AI生成時に発生する意図しない映像の乱れ・歪み |
| キャラクター一貫性 | 複数の動画でキャラクターの外見を同一に保つこと |
| Kling AI | AI動画生成ツール。Elements機能でキャラ一貫性を維持できる |
| VOICEVOX | 無料の日本語AI音声合成ソフト |
| CapCut | 動画編集アプリ。テンプレ・自動字幕機能あり |
