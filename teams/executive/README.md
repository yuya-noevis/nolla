# Nolla Executive Team — AI経営チーム

## 概要

Nollaの事業戦略・意思決定・実行管理を担うAI経営チーム。
最終意思決定は全てCEO（Yuya）が行い、AI経営メンバーは提案・分析・実行支援を担う。

## 直近の目標

**MVP完成 → 数名の熱狂的ユーザー獲得**

## メンバー

| 役職 | ディレクトリ | 責務 |
|------|-------------|------|
| **COO**（最高執行責任者） | `coo/` | 実行管理・スプリント運営・意思決定支援・資金調達準備 |
| **CPO**（最高プロダクト責任者） | `cpo/` | ユーザーリサーチ・MVP仕様定義・プロダクト開発チームへの橋渡し |
| **CMO**（最高マーケティング責任者） | `cmo/` | 初期ユーザー獲得・コンテンツ戦略・コミュニティ形成 |
| **臨床アドバイザー** | `clinical_advisor/` | 療育品質担保・エビデンスレビュー・コンテンツ監修 |

## 共有ファイル（`shared/`）

| ファイル | 内容 |
|----------|------|
| `principles.md` | 全メンバー必読の行動原則（忖度禁止・エビデンス主義等） |
| `weekly_review.md` | 週次レビューのテンプレート |

## 他チームとの関係

```
経営チーム（本チーム）
  ├── 戦略・方向性・ユーザー知見を提供
  │         ↓  CPOが橋渡し
  ├── プロダクト開発チーム（別ターミナル）
  │         具体的な設計・実装を担当
  │
  └── コンテンツ制作チーム（teams/content_production/）
            動画コンテンツの制作を担当
```

## 使い方

1. 各メンバーの `role.md` をシステムプロンプトとして読み込む
2. `shared/principles.md` は全メンバーが必ず参照する
3. 週次レビューは `shared/weekly_review.md` のフォーマットに従う
4. アウトプットは `outputs/` 配下に配置する
