# Long Paste — クリップボード読み込み

クリップボードの内容をpbpasteで取得し、作業に使用する。

## 手順

1. Bashツールで `pbpaste` を実行し、クリップボードの全内容を取得する
2. 取得した内容をユーザーに簡潔に要約して確認する（最初の数行と文字数）
3. 「この内容で何をしますか？」とユーザーに聞く

## 注意

- クリップボードが空の場合は「クリップボードが空です。先にテキストをコピーしてください」と伝える
- 内容の改変は行わない。取得したテキストをそのまま保持する
