## 0. エグゼクティブサマリー

AutiSpark は Nolla 最酷似コンセプトと言われるが、**単独スタートアップではなく、20年続くインド発の Mumbai ベース edutainment スタジオ「IDZ Digital / KidloLand」が 2020 年に投入したサブブランド**である。これは Nolla が想定していた「ASD 専用スタートアップ」像と大きく異なる。**「ASD 専用設計の独立企業」ではなく、「子ども向けゲーム工場が ASD ニッチに延伸した派生プロダクト」**と理解すべき。

最重要発見:
1. **未調達 (bootstrapped) で FY25 ₹120Cr (≈$14M USD) 売上、CAGR 58%、社員 215 名**。AutiSpark 単体ではなく KidloLand 本体の数字。
2. **創業者 Aditya & Nishant Mohatta (兄弟) に ASD の原体験は確認できず**。創業ストーリー型ではなく事業拡張型。
3. **140+ ゲームの大半は KidloLand の既存 IP・既存ゲームエンジン・既存制作パイプラインの転用・再目的化（リパッケージング）**と推定される（公式声明での否定なし、ゲーム傾向から強く推定）。これが「インド発・低コスト・大量提供」を可能にした最大要因。
4. **Sub $5/month (年 $59.99) のサブスク+大量無料ダウンロード+広告/IAP のフリーミアム**で米国・英語圏市場を獲得。RCT・大学共同研究などのエビデンス層は確認できず（Otsimo / Jade ND より弱い）。
5. **パッシブ計測層・ニューロテック層は皆無**。差別化軸ではない。Nolla が向かおうとしているニューロテック方向とは正面衝突しない。

### C-2. 「ASD 専用」と打ち出した理由・タイミング

公式マーケティングコピー（Google Play 説明文・公式サイト）から読み取れる訴求軸:

1. **OT / Special Educator 監修の権威付け** （個人名非公開、肩書のみ）
2. **140+ ゲーム + 1,000+ Worksheet** という量的優位
3. **「KidloLand の作り手」というブランド継承**
4. **Free Trial + 低価格年間サブスク** で意思決定ハードル低減

**ASD 専用カテゴリ自体が当時 (2020) はインド国内に競合が薄かった**（インドの先行 Avaz は AAC 特化、ASD 一般教育ゲーム集合体ジャンルは空白）。一方で米国市場には Otsimo (2014, トルコ発), Endless Reader 派生, Avaz Inc.（独立企業）等が既にいた。AutiSpark は**「Otsimo より安い・ゲーム数多い・印象的なグラフィック」**という量×価格のポジショニングで侵食していった可能性が高い。

### C-3. 最初のユーザー獲得方法（推定）

公式声明・インタビューが見つからないため、以下は推定:
- **KidloLand 既存 250M+ ユーザーへのクロスプロモーション**が最大の初期ブースター
- **Facebook / Instagram での親コミュニティ広告** (autispark の SNS は accountは継続的にレビュー・親の体験談を投稿)
- **Mom's Choice / NAPPA / Academics' Choice 等のアワード受賞でのブランディング**
- **ASO 最適化**：Google Play のアプリ ID が `com.iz.autispark.kids.autism.games.special.needs.educational.learning.therapy.social.skills.speech` という極端なロングテールキーワード詰め込み型 → ASO に対するテクニカルな投資が見える

### H-1. プロダクト批判（確認済）

## I. Nolla への示唆（最重要）

### I-1. 直接的に学べる点

1. **「ASD 専用」を打ち出すコピー戦略は機能する**：AutiSpark の Google Play DL 1M+ / 累計 3.6M+ は「ASD 専用」カテゴリ独占で取得した数字。Nolla も「発達障害・ASD・知的障害特化」を躊躇せず前面に出すべき。

2. **量的優位 (140+ ゲーム × 1,000+ ワークシート) が初期 PMF の鍵**：少数高品質より、初期は**幅広いゲーム集合体で「自分の子に合う 1 つを見つける」体験**を提供する方が、ターゲット層（多様な発達特性の子）にフィットする。Nolla の MVP 4 ゲームでは少なすぎる可能性。

3. **既存制作インフラの転用が低価格を可能にする**：AutiSpark は KidloLand のゲームエンジン・素材・パイプラインを流用。Nolla は新規開発のため**初期コスト構造で AutiSpark に勝てない**。差別化軸は価格戦争ではなく、エビデンス・パッシブ計測・年齢長期伴走に置くべき。

4. **多言語化は遅れて来る成長ドライバー**：AutiSpark は英語先行で、3-4 年後に Spanish / Portuguese / Indonesian / Turkish 追加。日本発の Nolla は逆に**日本市場 → 東アジア (韓・台・香) → 英語圏**の順で展開できる地理的メリットがある。

5. **アワード戦略が信頼構築に効く**：NAPPA / Mom's Choice / Academics' Choice / Google "Teacher Approved" の 4 認証セットを目標とすべき。Nolla の臨床アドバイザー監修と相性が良い。

6. **School Edition の独立 SKU 化**：AutiSpark は家庭向けと学校向けを別アプリにし、$119.99 買い切りで施設・自治体予算をキャプチャ。Nolla の ToB 展開時に踏襲すべきモデル。

7. **Bootstrapped でも黒字運営は可能**：21 年間未調達で 215 名・₹120Cr 売上を実現しているのは、**サブスク + 広告 + OEM 受託**の三本柱が機能している証拠。Nolla も無理に大型調達を急がず、KidloLand 型の小規模＋多 SKU で持続可能性を確保する選択肢がある。

### I-2. AutiSpark に欠けている領域 (Nolla の差別化チャンス)

| 領域 | AutiSpark | Nolla の差別化チャンス |
|---|---|---|
| **エビデンス層** | RCT・大学共同研究なし。OT 監修の権威付け止まり | RCT 設計・査読論文・脳波/視線データ等で**「測れる発達支援」**を訴求 |
| **パッシブ計測層** | 完全に不在 | 視線・反応速度・タップパターン・難度適応のログを保護者に可視化 |
| **個別最適化** | 弱い（複数プロファイル不在の指摘） | 適応型難度 (75-80% 成功率維持) を MVP 段階から実装 |
| **非言語児対応** | 音声指示依存で重度知的障害児にアクセシブルでない | 重度 ID + ASD を一次ターゲットに、文字なし・音声なし・アイコン主導で勝負 |
| **長期伴走** | 2-8 歳止まり | 3-18 歳の年齢ステージ全体に伴走、図鑑・コレクション・マイルームで継続性を作る |
| **エコシステム** | 単発アプリ | 親・施設・学校・研究機関の 4 ステークホルダー型の網 (Nolla の元設計と一致) |
| **療育エビデンスとゲームのつながり** | 「OT 監修」の信頼性のみ | NCI スコアリング・脳波測定・自治体療育記録連携など測定可能な裏付け |

### I-3. 「なぜインド発・低価格・低品質と言われながらも 3.6M DL を獲得できたか」のメカニズム解明

**検証された仮説**:
1. **既存制作インフラの再活用**で開発・運用コストが極小（KidloLand のゲームエンジン流用）→ サブスク $59.99/年でも黒字
2. **「ASD 専用」カテゴリは英語圏でも 2020 年時点で空白に近かった**（Otsimo の DL は 100K 規模）→ 先行者ではないが「量と価格」で十分追いつけた
3. **コロナ禍 (2020-2022) 家庭向けデジタル療育需要爆発のタイミング**にローンチが完全一致
4. **既存 KidloLand 250M+ ユーザーへのクロスプロモ + ASO 最適化 + Mom's Choice 等の認証**で、伝統的 PR 露出ゼロでも DL を伸ばした
5. **App Store / Google Play 経済圏の「アプリ単体課金で世界中の親が直接買える」**という構造的特性。インド発でも英語圏に届く。日本発の Nolla も同じレバレッジが効く。

**「低品質と言われながら」の部分の検証**:
- ビジュアル品質に関する明示的な「Otsimo より劣る」という第三者記事は確認できず（ユーザー指示の前提が要再確認）
- Educational App Store のような専門レビューサイトは AutiSpark を**「expert-approved, well-researched, engaging」**と肯定評価
- ただし**「heavy reliance on verbal instructions」「lacks personalization」「lacks visual progress rewards」**という構造的弱点は指摘されている → これは品質というより**設計思想の限界**

**結論**: AutiSpark の成功は「**低品質でも DL が取れた**」のではなく、「**既存資産を流用して低コストで提供しつつ、必要十分な品質と権威付け（OT 監修、アワード、KidloLand ブランド）を備えた**」結果である。Nolla がここから学ぶべきは「**品質を下げて勝つ**」ではなく「**既存資産の再利用で固定費を下げ、エビデンス層で上位差別化を取る**」ハイブリッド戦略。
