# NOLLA-001 「おはよう の うた」

## メタ情報
- 動画ID: NOLLA-001
- 尺: 60秒
- レベル: 入門（はじめて）
- 対象領域: 発語
- 療育アプローチ: 歌・リズムベース + インバーバル・フィルイン
- ターゲット行動: 「おはよう」の模倣発語
- サムネイル色: #A8D8B9（やさしいグリーン）+ ★

---

## タイムライン

---

### [0:00-0:03] オープニング（統一フォーマット）

- **映像**: クリームホワイト(#FFF8F0)の背景に、Nollaのメインキャラ「ノラ」が画面下からぴょんと登場。手を振る
- **音声**: 統一オープニングジングル（3音のベル音、マリンバ）
- **ナレーション**: （ゆっくり、歌うように）「ノラと いっしょ！」
- **SE**: ポン（柔らかい登場音）
- **演出メモ**: 毎回同じ登場アニメーション。キャラが画面中央に来たら手を振る。子どもに「始まった」と認識させる予測可能な導入

---

### [0:03-0:08] テーマ提示「おはよう」

- **映像**: 背景がクリームホワイトからやわらかい朝焼け色（薄オレンジ〜ピンクのグラデーション）に変化。おひさま（丸くて笑顔のキャラ）が画面上部にゆっくり昇ってくる。ノラがおひさまを見上げる
- **音声**: 穏やかなBGM開始（ウクレレ+マリンバ、95 BPM、Cメジャー）
- **ナレーション**: （ゆっくり、高めのトーンで語りかけ）
  「あさだよ。おひさまが のぼってきたよ」
  （1秒の間）
  「いっしょに おはよう って いおうね」
- **演出メモ**: おひさまの登場でテーマを視覚的に提示。ナレーションは1文ずつ間を取る。「おはよう」を初めて提示する瞬間——ノラが口を大きく開けて「お・は・よ・う」のスピーチモデリング

---

### [0:08-0:16] 歌パート1 + インバーバル①

- **映像**: ノラがおひさまに向かって手を振りながら歌う。背景に小さな雲がゆっくり流れる

- **歌**:（ゆっくり、明るいメロディ。ペンタトニック、Cメジャー）
  ```
  ♪ おひさま おひさま のぼったよ
  ♪ おは…
  ```
  → **ここで歌が止まる**

- **[0:12-0:16] 【インバーバル①】4秒間**
  - プロンプト: **音声**（ノラが「お・は・」まで口を動かし、口を開けたまま待つ）
  - 映像: ノラが口を大きく「よう」の形に開けて、期待の表情で画面を見つめる。口元をやや大きく描画
  - 音声: BGMフェードダウン（-20dB以下）。ほぼ無音
  - テロップ: 「おは___」（空欄部分がキラキラと光る）
  - **演出メモ**: 子どもが「よう」と言う時間。4秒は長く感じるがこれが療育効果の核。ノラの期待の表情が子どもの発話を促す

- **[0:16-0:18] 【強化①】**
  - 映像: ノラが満面の笑顔でジャンプ。小さなキラキラエフェクト（控えめに）
  - ナレーション:「おはよう！ じょうず！」（高めのトーン、喜びを込めて）
  - SE: キラキラ音（3音の上昇フレーズ）
  - **演出メモ**: 子どもが言えても言えなくても、ノラが正解を示して強化。罰的要素はゼロ

---

### [0:18-0:27] 歌パート2 + インバーバル②

- **映像**: シーンが変わり、ノラが小さなうさぎのキャラに出会う。うさぎがまだ眠そうにしている

- **ナレーション**: 「うさぎさんが いるよ。おはよう って いおう」

- **歌**:（同じメロディの繰り返し）
  ```
  ♪ うさぎさん うさぎさん おきてよ
  ♪ おは…
  ```
  → **ここで歌が止まる**

- **[0:24-0:27] 【インバーバル②】3秒間**
  - プロンプト: **視覚**（ノラが口を大きく開ける。音声プロンプトなし）
  - 映像: ノラが口を「よう」の形に大きく開け、うさぎを指さす。うさぎも目を開けて待っている
  - 音声: BGMフェードダウン。ほぼ無音
  - テロップ: 「おは___」
  - **演出メモ**: 2回目なのでプロンプトを1段階減らす。音声なし、口元の視覚手がかりのみ

- **[0:27-0:30] 【強化②】**
  - 映像: うさぎがパッと目を開けてニコニコ。ノラとうさぎが一緒に笑顔
  - ナレーション:「いえたね！ うさぎさん おきたよ！」
  - SE: ポン（柔らかい肯定音）
  - **演出メモ**: 「おはようを言ったら友だちが笑顔になる」という社会的結果を自然に提示

---

### [0:30-0:41] 応用パート + インバーバル③

- **映像**: シーンが変わり、ノラとうさぎが一緒に歩いていると、くまのキャラに出会う

- **ナレーション**: 「くまさんにも おはよう しよう！」
- **演出メモ**: ノラが手を振るジェスチャーを見せる（ビデオモデリング）

- **歌**:（同じメロディ、3回目の繰り返し）
  ```
  ♪ くまさん くまさん おはよう
  ♪ ……
  ```
  → **ここで歌が止まる（語頭から空ける）**

- **[0:38-0:41] 【インバーバル③】3秒間**
  - プロンプト: **ジェスチャー**（ノラが手を振るだけ。口元・音声のプロンプトなし）
  - 映像: ノラが手を振りながら、くまを見て笑顔で待つ
  - 音声: BGMフェードダウン。ほぼ無音
  - テロップ: 「______」（全部空欄）
  - **演出メモ**: 3回目。ハードルを上げて「おはよう」全体を子どもに任せる。ジェスチャーが唯一の手がかり

- **[0:41-0:45] 【強化③】**
  - 映像: くまがガバッと起き上がり、両手を広げてハグのポーズ。ノラ・うさぎ・くまが3人で笑顔
  - ナレーション:「すごい！ くまさんも げんきに なったよ！」
  - SE: キラキラ音 + ポン
  - **演出メモ**: 3人に増えることで「みんなで」の楽しさを演出。達成感を高める

---

### [0:45-0:52] 振り返り + インバーバル④

- **映像**: ノラ・うさぎ・くまが並んで画面を見つめる（=視聴者に向かって）。背景はおひさまが明るく輝いている

- **ナレーション**: （ゆっくり、語りかけ）
  「みんなに おはよう できたね。」
  「さいごに いっしょに いおう。せーの…」

- **[0:48-0:52] 【インバーバル④】4秒間**
  - プロンプト: **なし（自発待ち）**
  - 映像: ノラ・うさぎ・くまが口を開けて待っている。3キャラ全員が期待の表情
  - 音声: 完全な無音（BGMもなし）
  - テロップ: なし（視覚プロンプトも外す）
  - **演出メモ**: 最大のチャレンジ。プロンプトゼロで子どもの自発的発語を待つ。入門レベルではここで言えなくても全く問題ない

- **[0:52-0:57] 【強化④】最大強化**
  - 映像: 3キャラが一斉に「おはよう！」と言い、ジャンプ。おひさまもキラキラ。画面全体がやわらかい光に包まれる（フラッシュではなく、ゆっくりした光のグラデーション変化）
  - ナレーション:「おはよう！ できたね！ すごいね！」
  - 歌:（最後のフレーズ）
    ```
    ♪ おはよう おはよう げんきだよ
    ♪ おはよう おはよう うれしいね ♪
    ```
  - SE: 拍手音（パチパチ、3-4回）+ キラキラ音
  - **演出メモ**: 動画内で最も盛り上がるポイント。ただし音量急変なし。キラキラエフェクトも穏やか。「言えても言えなくても、みんな一緒におはよう」の包括的な肯定

---

### [0:57-1:00] エンディング（統一フォーマット）

- **映像**: ノラが画面中央で手を振る（オープニングと対称）。背景がクリームホワイトに戻る
- **音声**: 統一エンディングジングル（オープニングと同じ3音のベル、少しテンポを落として）
- **ナレーション**: （ゆっくり）「またね。ノラと いっしょ！」
- **SE**: ポン（柔らかい終了音）
- **演出メモ**: オープニングと対になる構成。「手を振って終わる」が毎回の統一フォーマット。ループ視聴の場合、エンディング→オープニングが自然につながるようBGMのキーを統一

---

## 療育設計セルフレビュー

Agent 02（療育設計）の視点で台本を検証:

| チェック項目 | 判定 | メモ |
|---|---|---|
| ターゲット行動が反映されている | ✅ | 「おはよう」の模倣発語が全体の核 |
| 語彙がリスト範囲内 | ✅ | おはよう、おひさま、あさ、げんき、うさぎ、くま。新出は「おはよう」「げんき」の2語 |
| インバーバル4回配置 | ✅ | 0:12, 0:24, 0:38, 0:48 に配置。療育目標シート通り |
| プロンプト・フェイディング | ✅ | 音声→視覚→ジェスチャー→なし の4段階 |
| 間の秒数 | ✅ | 4秒→3秒→3秒→4秒。入門レベルの3-5秒範囲内 |
| 強化が行動直後 | ✅ | 各インバーバル直後に即時強化 |
| 受動視聴の回避 | ✅ | 30秒あたり2回のインバーバル（合計4回/60秒） |
| 般化への配慮 | ✅ | おひさま→うさぎ→くま→視聴者 の4場面で使用 |
| 予測可能な構成 | ✅ | 統一オープニング/エンディング。歌の繰り返し構造 |
| 否定表現の排除 | ✅ | 全て肯定的な表現。「できたね」「じょうず」「すごい」 |
| 専門用語の排除 | ✅ | 台本内に専門用語なし |
| 1動画1テーマ | ✅ | 「おはよう」のみ |

**レビュー判定: 承認 → Step 3へ**
