# Nolla SNS戦略レポート — 「今日もわが家は通常運転」

2026-04-01作成

## 1. ターゲットの深層心理

### フラストレーション Top 5

| 順位 | 内容 |
|---|---|
| 1 | **「誰にもわかってもらえない」孤立感** — 定型発達児の親とは悩みの次元が違い、発達障害児の親同士でも特性が違って本音を言えない |
| 2 | **毎日の癇癪・パニック・こだわりへの対応疲れ** — 一つ一つは小さいが365日休みなく続く慢性消耗。「疲れた」と思う自分を責める二重ストレス |
| 3 | **パートナー（夫）の無理解・温度差** — 「普通でしょ」「気にしすぎ」。ワンオペで「一緒に家族をやってるのにひとりぼっち」 |
| 4 | **園・学校・周囲からの排除体験** — 入園拒否、加配がつかない、他の保護者の冷たい視線。常に「迷惑かけてるかも」という罪悪感 |
| 5 | **制度・手続きの複雑さと情報の分断** — 受給者証に1〜2ヶ月、自治体で手続きが違う、療育の空きがない |

### 願望・欲求 Top 5

| 順位 | 内容 |
|---|---|
| 1 | **「この子はこの子のままで大丈夫」と心から思えること** — 受容は一度で完了せず日々揺れ動いている |
| 2 | **自分自身の時間と精神的余裕** — ASD児の母親の約40%が抑うつ症状。セルフケアは贅沢でなく生存戦略 |
| 3 | **同じ立場の人とつながり本音を言える場所** — ただし既存コミュニティは重い。「気軽に覗ける軽さ」が求められている |
| 4 | **子どもの将来の自立への道筋が見えること** — 「親亡き後どうなる？」への具体的な選択肢 |
| 5 | **「頑張ってるね」と認めてもらうこと** — アドバイスより先にまず肯定されたい |

### 恐れ・不安 Top 5

1. **「親亡き後」問題** — 自分が死んだらこの子はどうなるのか
2. **二次障害（不登校・うつ・自傷）** — 「今の対応が間違っていたら取り返しがつかない」
3. **社会からの偏見と無理解** — 「しつけが悪い」「親のせい」
4. **きょうだい児への影響** — 我慢させてしまっている罪悪感
5. **自分自身が壊れること** — 倒れたら子どもを見る人がいない

### 共感トリガー（反応率が高いシーン）

1. スーパーで寝転がって泣き叫ぶ子を冷たい目の中で抱え上げる
2. 今日の晩ごはん、白ご飯しか食べなかった
3. 園の先生に呼び出された帰り道の車の中で泣いた
4. 小さな成長を喜べる人が周りにいない
5. 定型発達の子の「普通の愚痴」を聞くのがつらい
6. 夜、寝かしつけた後に「今日も怒りすぎた」と自己嫌悪する
7. 「この子の将来」と検索して、余計に不安になって画面を閉じる

### SNS行動パターン

- **ゴールデンタイム：21〜23時**（寝かしつけ後）。最も感情が動きやすい。共感コンテンツへの反応が高い
- **サブタイム：10〜14時**（子が園に行っている間）。情報収集モード
- **危機時**：癇癪直後、園から指摘された直後に衝動検索
- **1回あたり15分未満が8割**

### 信頼構築の条件

- **当事者性のリアリティ** — 「この人、本当にわかっている」と感じる具体的な描写の解像度
- **押し付けない姿勢** — 「こうすべき」ではなく「うちはこうしてる」
- **特性描写の正確さ** — ズレると「わかってない人」と見抜かれる
- **ユーモアと温かさのバランス** — 重い体験談ばかりだと「見ると疲れる」で離脱
- **商業臭の薄さ** — 「障害を商売に利用されている」感に非常に敏感

## 3. スクロールを止める投稿設計

### タイトルカード（1枚目）の最適化

**5つのパターン：**

| パターン | 仕組み | 例 |
|---|---|---|
| A. 「え、うちだけ？」型 | 孤立不安を刺激、答えを知りたくてタップ | 「え、靴下のつま先の縫い目で泣くの、うちだけ？」 |
| B. 瞬間切り取り型 | 具体的すぎて「わかる！」が即発動 | 「スーパーの蛍光灯で固まった」 |
| C. 子どもの名言型 | 予測不能で次が気になる | 「"ママ、この音やめて"（冷蔵庫の音）」 |
| D. ビフォーアフター落差型 | 期待と現実のギャップがクスッを生む | 「"今日は大丈夫かも" → 3分後」 |
| E. 数字で具体化する型 | 数字は目を止める力が強い | 「同じ動画、今日47回目」 |

**現在のタイトルへの改善提案：**

| 現タイトル | 課題 | 改善案 |
|---|---|---|
| 「いつもの道じゃないとダメ」 | 「ダメ」がネガティブ、主語が曖昧 | 「"近道見つけた！" → 足が止まった」 |
| 「タグがチクチクする」 | 一般的すぎる | 「タグがチクチクする → 全部切った → まだチクチクする」 |
| 「掃除機が怖い」 | 共感より同情に寄る | 「掃除機の音で別の部屋に避難する息子」 |

> **共通ポイント**: 現在のタイトルは「特性の説明」。「特性が起こした"瞬間"」に変えると画が浮かんで止まる。**0.7秒以内に読めて、画が浮かぶか**が勝負。

### コメントを誘発する仕掛け（CTA以外）

1. **二択クイズ** — 「ここでお母さんはどうした？ A: 諦めた B: もう一回チャレンジ」（認知負荷が低く回答率がオープンQの3倍）
2. **未完成のオチ** — 「…で、どうなったと思う？」で切る。次の投稿で答えを出すとフォロー継続率もUP
3. **超具体的な描写** — 抽象的→「わかる」で終了。具体的→「うちは○○だった」と自分の話がしたくなる
4. **軽い問いかけ（独り言風）** — 「みんなの"これだけは絶対"ってなんだろう…」（命令形CTA避ける）
5. **投稿直後30分のコメント即レス** — アルゴリズムは初速30分を最重視。質問で返すと往復が増える

### 保存される投稿の特徴

1. **後で見返す実用情報** — Tips枚を「冷蔵庫に貼りたいレベル」の具体性に
2. **「誰かに見せたい」内容** — Tips冒頭に「パパ（おばあちゃん/先生）にも見せて」
3. **シリーズ化** — 特性別シリーズにすると過去分もまとめて保存される
4. **情報密度が高いが読みやすい** — 箇条書き、「なぜ→こうする→これはNG」構成
5. **感情的に「お守り」になる** — 辛いときに見返して元気が出るトーン

## 4. 投稿タイミングと形式

### 推奨スケジュール

| 曜日 | 時間 | 内容 |
|---|---|---|
| 月曜 | 20:30 | あるある劇場（カルーセル） |
| 木曜 | 20:30 | ②③④⑤のいずれか（カルーセル） |
| 土曜 | 20:30 | 反応が良かった投稿のリール化 |
| 毎日 | 15:00 | ストーリーズ（アンケート・裏話・次回予告） |

> 20:30〜21:00が**ゴールデンタイム**（寝かしつけ後）。投稿後30分はコメント返信に常駐。

### カルーセル vs リール

| | カルーセル | リール |
|---|---|---|
| リーチ率 | 約14% | 約31%（2.2倍） |
| エンゲージメント率 | **10.15%** | 6% |
| 用途 | **主力**（あるある×Tips） | **リーチ拡大用** |

- **カルーセル = 本業**（週2-3回）。保存率高、フォロワーとの関係深化
- **リール = 新規獲得用**（週1回）。人気投稿を15-30秒スライドショー化（BGM付き）

### 2026年アルゴリズムの重要変更

1. **DM送信（Sends）が最重要シグナル** — 「誰かに送りたくなる」設計が鍵
2. **94%がAIレコメンド** — フォロワー以外に刺さるかが成長のカギ
3. **ハッシュタグの効果低下** — キャプション内のキーワード（感覚過敏、こだわり等）を自然に入れるSEO的アプローチが有効
4. **カルーセルが「チートコード」** — スキップされても2枚目が別タイミングで再表示される
5. **オリジナルコンテンツ優遇** — イラストは本質的にオリジナルなので構造的に有利

## 5. 現在の投稿構成への改善提案

### 現構成の強み
- **ストーリー＋Tipsのハイブリッドが秀逸**（感情で引き込んでから知識を渡す）
- **フィクションキャラが正解**（実録系だらけの発達障害ジャンルでブルーオーシャン。「笑っていいのか」問題を回避）
- **カルーセルが2026年アルゴリズムと相性抜群**

### 追加すべきこと

1. **1枚目の「引き」を強化** — 特性の説明→「瞬間の切り取り」に変える
2. **あるある以外のピラーを追加** — ③成長メモ、④ホンネ、⑤制度情報でフィードに変化を
3. **曜日固定のシリーズ感** — フォロワーの習慣化を促す
4. **CTAをピラー別に使い分け** — あるある→シェア促進 / Tips→保存促進 / ホンネ→コメント促進
5. **ストーリーズ連携** — 「次回のテーマどっち？」アンケート、制作裏話でフォロワー親密度UP
