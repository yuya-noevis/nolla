# PiAPI（piapi.ai）安全性・信頼性調査レポート

**調査日**: 2026年3月24日
**対象サービス**: PiAPI（https://piapi.ai）

---

## 1. 運営会社情報

### 基本情報
- **サービス名**: PiAPI
- **公式URL**: https://piapi.ai
- **設立年**: 2024年（最近設立）
- **形態**: AI API統合プラットフォーム
- **所在地**: 非公開（公式情報では明記されていない）
- **連絡先**: contact@piapi.ai

### 会社の透明性
**懸念点**:
- 運営会社の所在地・企業名・法人登記情報が公開されていない
- Crunchbaseに登録されているが、詳細情報は限定的
- 公式ウェブサイトに「About Us」や「Team」ページが見当たらない
- 創業者・経営陣の情報が公開されていない

**評価**: ⚠️ **低い**（営利企業として、会社情報の透明性が不十分）

## 2. ユーザーレビュー・評判

### 肯定的な評判
- **開発体験**: 「ワークスペースとウェブフック機能が優れている」「統合設定が簡単」との声
- **画像生成機能**: 「Flux APIは世界中の開発者に人気」と評価
- **コスト効率**: 「複数アカウントを接続しやすく、開発時間を削減」
- **ゲーム・AR/VRアプリ開発**: マーケティングやクリエイティブ用途での採用事例あり

### レビュープラットフォームの現状
- **Trustpilot**: 詳細なレビューが見当たらない（Pi.aiと混同している可能性）
- **Reddit**: 有意な議論・評判情報が確認されない
- **ProductHunt**: 特定の詳細レビューが検出されない
- **評価サイト**: Crozdesk、AI Founder Kitなどで紹介されているが、評価数は少ない

**評価**: ⚠️ **中程度**（肯定的な声はあるが、第三者評価プラットフォームでの実績が限定的）

## 3. セキュリティ面

### 実装状況
- **HTTPS**: 公式サイトはHTTPSで保護されている
- **セキュリティプロフィール**: Nudge Security の認定セキュリティプロフィール掲載あり
- **プライバシーポリシー**: 存在することは確認されているが、詳細内容は公開ウェブサイトで容易にアクセスできない

### 懸念点
- **プライバシーポリシーへのアクセス**: 公式サイトから直接リンクされていない、または見つけにくい
- **データ保護方針**: 詳細な説明が公開されていない
- **GDPR/国際基準への対応**: 明確な対応状況の説明がない
- **ユーザーデータの取扱い**: 第三者（AI企業）への転送・保持期間について不明確

**評価**: ⚠️ **要確認**（基本的なセキュリティはあるが、プライバシー面で透明性が不足）

### 推奨アクション
- contact@piapi.ai に直接問い合わせて、プライバシーポリシーと個人情報保護方針の詳細を入手
- データがどのように保存・使用されるかを確認必須

## 4. 料金の透明性

### 価格体系

| プラン | 月額料金 | 付与クレジット | 概要 |
|--------|---------|----------------|------|
| **Free** | $0 | $0 | 基本機能、APIアクセス限定 |
| **Creator** | $15 | $10/月 | 無制限タスク処理 |
| **Pro** | $60 | $60/月 | すべてのタスクタイプ |
| **Enterprise** | $100 | $0 | チーム向け、3倍の並行処理 |

### 課金構造の問題点

**⚠️ 重大な懸念: 二重課金モデル**

1. **サブスクリプション料金**: 月額$15～$100（Creatorプランで例示）
2. **API使用量料金**: サブスクリプション内のクレジット（$10/月）を超えた利用分は別途購入が必要
3. **アカウント管理料**: 別途 $5～$10/月/シート

**隠れた費用の例**:
- Creator($15/月)で$10クレジット付与 → 実質$5分不足する
- 多くの開発者は追加クレジットの購入が必須になる可能性が高い
- クレジットの有効期限: **180日** → 未使用分は失効

### 透明性の評価

**「隠れた課金はない」という公式主張**:
> 「生成時間と実際に必要な機能にのみ料金を支払う — 隠れた料金なし」

**実態**:
- 技術的には「隠れた」課金ではないが、**デュアル課金(サブスク+従量)** の構造が複雑で、初期段階では追加費用が発生することが明確でない
- 「透明」というより「複雑」と評価すべき

**評価**: ⚠️ **やや不透明**（二重課金構造の説明が不十分、初期コストが予測しにくい）

## 6. リスク・懸念点の整理

### 高リスク（⚠️⚠️⚠️）

1. **企業情報の非透明性**
   - 所在地、法人登記、経営陣が非公開
   - スタートアップながら情報開示が不十分
   - 詐欺・バックドア対応の疑い払拭できず

2. **プライバシー・データ保護の不明確さ**
   - ユーザーのAPI呼び出し内容（入力プロンプト・生成画像等）の取扱いが不明確
   - 第三者AI企業（MidjourneyやOpenAIなど）へのデータ流出リスク
   - 個人情報や機密情報の送信は**推奨できない**

3. **利用規約・法的責任が不明確**
   - ToS（利用規約）の詳細アクセス性が低い
   - 突然のサービス終了・アカウント削除のリスク

### 中リスク（⚠️⚠️）

4. **運営資金・経営の持続可能性**
   - Crunchbaseに登録されているがファンディング情報が限定的
   - 2024年設立の新興企業であり、経営持続性が未検証

5. **カスタマーサポートの品質**
   - メインサポートは Discord
   - contact@piapi.ai は「collaboration inquiries only」（協業問い合わせのみ）
   - 一般ユーザーのサポート対応が充実していない可能性

6. **料金システムの複雑性**
   - 二重課金モデルにより、総コストが予測しにくい
   - 新規ユーザーにとって「思ったより高い」になる可能性

### 低リスク（⚠️）

7. **Midjourney API サービス終了**
   - 公式情報: 「We no longer provide Midjourney API」
   - 既存ユーザーは「legnext.ai」への移行を推奨
   - サービス継続性に関する懸念の材料

## 8. 使用上の推奨レベル

### ✅ 推奨できる用途
- **非機密のプロトタイプ開発**: 学習・実験目的での画像/動画生成
- **個人プロジェクト**: 社外秘情報を含まない範囲での利用
- **低リスク業務**: マーケティング資料の素材生成など

### ⚠️ 慎重が必要
- **本番環境の商用利用**: サービス停止・突然の利用規約変更リスク
- **顧客データ処理**: ユーザーの個人情報を含むプロンプト送信
- **機密情報の使用**: 企業秘密・知的財産を含む依頼

### ❌ 推奨しない
- **医療・金融・法律情報の処理**: 規制対象データの送信
- **PII（個人識別情報）を含むデータの処理**: GDPR/個人情報保護法との抵触リスク
- **監査必須ビジネス**: SOC 2/ISO 27001など、公式認証がない

## 9. 推奨される次のステップ

1. **直接問い合わせ**
   ```
   contact@piapi.ai に以下を問い合わせ:
   - 会社の法人登記情報・所在地
   - プライバシーポリシー全文
   - データ保護・GDPR対応状況
   - SLA（サービスレベル契約）
   - 監査・セキュリティ認証の有無
   ```

2. **代替サービスの検討**
   - **公式API直接利用**: Midjourney / Flux / Suno 公式 API
   - **認証済みの統合プラットフォーム**: RunwayML、ElevenLabs等（セキュリティドキュメント公開済み）

3. **試行段階でのテスト**
   - Free プランで小規模にテスト
   - 機密情報は一切含めず、非本番環境のみで利用
   - 実装前に利用規約を全文確認
