# Nolla 競合分析マトリクス (2026-04-22 更新)

## 3大ベンチマーク競合（Nolla の最優先観察対象）

| 競合 | 位置づけ | 最大の特徴 | Nolla との距離 |
|---|---|---|---|
| **Jade ND（Jade Autism）** | **Nollaアイデアのベース・最直接競合** | ゲームベース+8ドメイン能力可視化+1,500ゲーム。**160K+ DL・175カ国・4,000 students・479 educators**（公式 jadend.tech） | **最も近い。機能構造が酷似** |
| **Otsimo** | **多角展開の直接競合** | 4プロダクト（Special Education / Speech Therapy / AAC / for Schools）。100万+ユーザー・93カ国 | **複数領域で並走** |
| **Dubu（旧DoBrain）** | **先行失敗事例の参考** | 20,000モジュール・60万ユーザー・韓国B2G 8,000校。**主要RCT失敗（PMC9806509）** | **直接競合ではないが戦略教訓が最重要** |

**3社共通の構造的脆弱性**: 主力アプリの**査読RCTが存在しない**（Dubuのみ実施済だが主要評価項目で有意差なし）。Nolla の最大の差別化余地はここにある。

## 製品別詳細分析

### 国内製品分析

#### 1. LITALICO発達ナビ
- **強み**: 発達障害情報の最大ハブ（会員50万人）、親向けコミュニティ
- **弱み**: アプリではなく情報プラットフォーム、療育機能なし、ゲーム性なし
- **価格**: 基本無料、プレミアム月額制
- **Nollaとの差別化**: Nollaはゲーム型療育。LITALICOは情報・相談型。補完関係の可能性

#### 2. CoDMON
- **強み**: 施設・親向けの統合管理システム、出欠・発達記録の一元化
- **弱み**: 施設向け、児童本人向けゲーム機能なし、認知測定なし
- **価格**: SaaS（施設単位）
- **Nollaとの差別化**: CoDMONは施設運営ツール。Nollaは家庭向けゲーム療育

#### 3. たすくシリーズ
- **強み**: ASD児向けの絵カード・スケジュール管理、支援現場での実績
- **弱み**: ゲーム性低い、単機能、認知測定なし、適応型難度なし
- **価格**: 500-2000円の買い切り
- **Nollaとの差別化**: たすくは支援ツール。Nollaは「ハマるゲーム」

#### 4. ワオっち！
- **強み**: 2-8歳向けの知育ゲーム、定型発達児のベンチマーク
- **弱み**: 障害児向けに特化していない、難度調整が限定的
- **価格**: 基本無料+課金
- **Nollaとの差別化**: ワオっちは定型発達向け。NollaはID+ASD特化

#### 5. 育ちの舎アプリ
- **強み**: 発達支援アプリの先進例
- **弱み**: 情報少ない、規模不明、認知測定機能不明
- **価格**: 詳細不明
- **Nollaとの差別化**: 詳細比較不可（公開情報限定）

> **【2026-04-26 重大訂正】** 初版で「ポルトガル・アヴェイロ大卒業」「185K DL/179カ国/347校/5,000学生」と記載していたが全て誤情報。下記訂正済み正確情報。出典: jadend.tech / uvv.br / patient-innovation.com

- 創業者: **Ronaldo Cohin**（自閉症の息子を持つ親、UVV Computer Science 卒業生）
- 規模（公式 jadend.tech 2026-04時点）: **160,000+ DL、175カ国、4言語、4,000 students、479 educators**

**プロダクトラインアップ**:

| プロダクト | 内容 |

| **Jade Edu（B2B）** | 教師向け管理、ABA段階設定、年額/生徒サブスク（**「347校導入」は出典不明・要確認**。公式 jadend.tech 表記は「4,000 students、479 educators」） |

**能力測定 8ドメインモデル**（Nolla の NCI と直接競合する設計思想）:

ゲーム内のユーザー選択・反応時間・エラーパターンからリアルタイム評価し、適応型難度自動調整。

- **強み**:
  - **Nolla の設計思想と最も近い**（ゲームベース+能力可視化+適応型難度）
  - 1,500ゲームの圧倒的コンテンツ量
  - 4プロダクトのエコシステム
  - B2B学校チャネル（公式：479 educators・4,000 students）
  - ビジュアル基盤で言語依存性を最小化
- **弱み**:
  - **主力アプリ Jade App の査読RCT論文ゼロ**（Jade Astea 視線追跡のみ論文あり）
  - 対象年齢が **3-11歳で頭打ち**（12-18歳カバーなし）
  - 日本未進出、日本語未対応
  - 財務・資金調達情報が非公開
  - ブラジル・中東中心で先進国医療への食い込み弱い
- **Nollaとの差別化**:
  - **Nolla**: 3-18歳フルカバー + 日本語深化 + RCTエビデンス先取り + 日本医療/教育制度統合
  - Jade ND: 3-11歳限定 + 多言語グローバル + RCT未実施 + ブラジル/中東拠点

## 競合分析から導き出す戦略的示唆

### 1. 差別化戦略：「認知科学 ✕ ゲーム設計」

**Nollaの独自価値提案（UVP）:**
- CogniFitの「認知測定の科学」 + Toca Bocaの「ゲーム設計の卓越性」 + Goallyの「親介入の実装度」を結合

**市場ポジショニング:**
- 「療育アプリ」ではなく「ASD児がハマるゲーム（療育効果付き）」
- 親向けスローガン: 「毎日のゲームが、認知機能の個別化トレーニングに」

### 2. 認知測定の深化（RCT実施）

**現在の課題**: CogniFitに学術的エビデンス量で負けている

**対抗策**:
- MVP実装後、半年以内にRCT設計着手
- 測定軸: 成功率、習熟速度、NCI推移の個別化効果
- 論文公表で「知的障害特化型の臨床効果」を実証

### 3. 親連携機能の段階化

**Phase 1（MVP）**: NCI型スコアリング + ダッシュボード表示  
**Phase 2**: トークン報酬の親への通知 + 現物報酬連携オプション  
**Phase 3**: 施設向けダッシュボード（CoDMON等との連携）

### 4. AAC機能の除外判断

**決定**: MVP段階では言語療法機能を除外（正解）

**理由**:
- Speech Blubsがこの領域で強い
- AAC は SLP 資格や医療管理が必要（スコープ外）
- NCI + ゲーム性に注力すべき

**後期対応**: v2以降、ロゴペディア連携で検討（自社実装でない）

### 5. マーケット拡張方向

**国内市場**:
- LITALICO との連携（情報→ゲーム流入）
- 放課後等デイサービス向けパッケージ（CoDMON連携）

**国際市場**:
- 英語版でグローバル展開（2-3年後）
- Otsimoより**学術的に優位な立場**で攻撃

## 最終結論

### Nollaの市場ポジション

```
従来型療育アプリ（たすく等）    ← 単機能、ゲーム性欠
                ↓
         Nolla（MVP）← 認知測定＋ゲーム性
                ↓
認知トレーニングアプリ（CogniFit） ← 科学的だが障害特化でない
```

**Nollaが占める空白**: 「知的障害+ASD特化 ✕ 認知測定 ✕ ゲーム性」

**成功要件**:
1. MVP 実装品質（Toca Bocaレベルの楽しさ）
2. NCI スコアリングの実装検証（CogniFitレベルの信頼性）
3. RCT による臨床エビデンス
4. 親向けダッシュボードの UX（Goallyレベルの実用性）

全てが揃えば、**独占市場（IQ21-110の療育ゲーム）** を形成可能。
