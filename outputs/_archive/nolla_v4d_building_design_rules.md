# Nolla v4d 建物デザインルール（厳守・定量版）

**作成日**: 2026-04-04（全面改定）
**絶対基準画像**: v4d_castle_bg.png
**出力形式**: 16:9 ランドスケープ、2K解像度（1456x816）

全ての生成画像はこのドキュメントの数値に寸分の狂いなく従うこと。

---

## 1. キャンバス構成（px値は2K基準。比率で適用）

```
┌─────────────────────────────────────────────┐ ← 画面上端 (0%)
│                                             │
│              空ゾーン                        │
│         (グラデーション + 雲/月/星)           │
│                                             │
│     ┌─────────┐                             │
│     │サイド建物│  ┌───────────────┐  ┌──┐   │
│     │ (左)    │  │               │  │サ│   │
│     │h:25-30% │  │  メイン建物    │  │イ│   │
│     │w:10-13% │  │  h:65-75%    │  │ド│   │
│     └─────────┘  │  w:35-50%    │  │建│   │
│                  │  水平中央     │  │物│   │
│                  │               │  │右│   │
│                  └───────────────┘  └──┘   │
├─────────────────────────────────────────────┤ ← 地面上端 (96%)
│            地面ストリップ h:3-4%             │
└─────────────────────────────────────────────┘ ← 画面下端 (100%)
```

---

## 2. 地面ストリップ（厳密固定）

| 項目 | 値 |
|------|-----|
| 高さ | 画面高さの **3〜4%**（25〜33px） |
| 色 | ダークグレー `#4A4A4A`〜`#555555` フラット単色 |
| 上端ライン | 完全な水平直線。起伏なし |
| 質感 | 無地。テクスチャ・模様・装飾なし |
| 下端 | 画面下端と一致 |

**禁止**: 草・花・道・石畳・影・模様。地面は薄いフラットなグレー帯のみ。

---

## 3. メイン建物（厳密固定）

### 配置

| 項目 | 値 |
|------|-----|
| 水平位置 | **画面幅の50%地点に中心軸** |
| 底面 | 地面ストリップ上端に接地 |
| 頂点マージン | 画面上端から **5〜10%** の余白 |

### サイズ

| 項目 | 縦長タワー型 | 標準型 | 横広型 |
|------|------------|--------|--------|
| 高さ（画面高さ比） | 70〜75% | 65〜70% | 60〜65% |
| 幅（画面幅比） | 25〜35% | 35〜45% | 40〜50% |
| 例 | 時計塔、灯台 | ファクトリー、天文台 | 城、本 |

### ボクセル仕様

| 項目 | 値 |
|------|-----|
| ブロックサイズ | 2〜4px四方（2K基準） |
| 曲面表現 | ブロックの段差（ステップ）で表現 |
| アンビエントオクルージョン | あり（接合部に柔らかい影） |
| シェーディング | スムース（光のグラデーション） |
| フォーカス | **常にシャープ**。ブラーなし |

---

## 4. サイド建物（最重要セクション）

### 基本原則: メイン建物との世界観連動

**サイド建物はメイン建物と同じ世界観・テーマの建物にする。汎用的なカラフルブロックは禁止。**

| メイン建物のテーマ | サイド建物の例 |
|------------------|-------------|
| 中世の城 | 中世の村の家（木骨造り、尖った屋根、石壁） |
| ファクトリー/工場 | 小さな倉庫、煙突付き工房 |
| 天文台/宇宙 | 小さなドーム屋根の観測小屋、星型の装飾付きの家 |
| 本/図書館 | 小さな本屋、ペン型の塔 |
| 風車 | 小さな農家、穀物倉 |
| 和風寺院 | 小さな茶屋、灯籠付きの家 |
| 温室カフェ | 小さな花屋、ケーキ屋 |
| オルゴール/音楽 | 小さなレコード屋、楽器店 |
| 気球ステーション | 小さなチケット小屋、地図屋 |

### 配置（v4d_castle_bg.png 実測値。厳守）

| 項目 | 左サイド | 右サイド |
|------|---------|---------|
| 露出幅（画面幅比） | **5〜6%** | **3〜5%** |
| 高さ（画面高さ比） | **15〜18%** | **12〜15%** |
| メイン建物との高さ比 | メインの **1/5〜1/4** | メインの **1/6〜1/5** |
| 底面 | 地面ストリップ上端に接地 | 地面ストリップ上端に接地 |
| 画面端からの距離 | 0px（画面端に接する） | 0px（画面端に接する） |

**サイド建物は背景要素。ほとんど見えないレベルに小さく控えめにする。**
左が右より少しだけ大きい（非対称）。

### 構成

| 項目 | 値 |
|------|-----|
| 建物の数（各サイド） | **2〜3棟**が隣接・重なり合う |
| 建物の形 | メインと連動したテーマの小さな建物。ほぼシルエットレベル |
| 窓 | 各棟に1個程度の極小窓。窓から暖色の光がほんのり |
| 装飾 | ほぼなし。屋根の形でテーマ連動を示す程度 |
| 色 | メイン建物と同系統。控えめ |

### レンダリング

| 項目 | 値 |
|------|-----|
| 被写界深度ブラー | **強め**（ガウシアンブラー半径6〜10px相当） |
| ボクセルスタイル | メインと同じだがブラーでほぼ判別不能 |
| 明度 | メインより **15〜20%暗め** |
| 存在感 | **雲と同程度かそれ以下**。背景に溶け込む |

### 禁止事項
- サイド建物が目立つ・主張する
- サイド建物の露出が画面幅の6%を超える
- サイド建物の高さがメインの1/4を超える
- サイド建物がシャープに見える
- サイド建物のディテールが読み取れるほど大きい
- メインと無関係なデザイン
- 左右どちらかだけ、またはサイド建物なし

## 5. 空のデザイン

### グラデーション
- 方向: **上→下の縦グラデーション**固定
- 色: 建物ごとに異なる空の色にする。同じ空を使い回さない
- 遷移: なめらか。バンディングなし
- 建物の色と被らない色にする

### リファレンスパレット（例示。自由に調整OK）

| 時間帯 | 上部 | 下部 | 雰囲気 |
|--------|-----|------|--------|
| 早朝 | 薄い紫〜ラベンダー | ソフトピンク〜クリーム | 静かで優しい |
| 朝 | 明るい水色 | 薄い黄〜クリーム | 爽やか |
| 昼 | 鮮やかな青 | 薄い青〜白 | 元気 |
| 午後 | やや深い青 | ピーチ〜薄いオレンジ | 穏やか |
| 夕方 | ピーチサーモン | クリーム〜薄いゴールド | 温かみ |
| マジックアワー | 紫〜ピンク | オレンジ〜サーモン | 幻想的 |
| トワイライト | 深い青紫 | ソフトピンク〜ラベンダー | 神秘的 |
| 夜 | 深い紺〜ネイビー | 暗い紫〜インディゴ | 落ち着き |

## 6. 雲

### 共通仕様

| 項目 | 値 |
|------|-----|
| 形状 | ぽってりした丸い塊。2〜3個の球が横に重なった形 |
| エッジ | ソフト。輪郭線なし |
| サイズ | 画面幅の **5〜10%** |
| 個数 | **2〜4個**。奇数配置推奨 |
| 配置 | 画面上部20〜60%の範囲。建物の後ろにも前にもOK |
| 左右バランス | 非対称 |
| レンダリング | **スムース**（ボクセルにしない） |
| 色 | 空の色に影響される（夕焼け=暖色寄り、昼=白+グレー影） |

### 空の明るさ別

| 空 | 雲の色 | 個数 |
|----|--------|------|
| 明るい空 | 白+底面にグレー影 | 3〜4個 |
| 暖色空 | 暖色に染まった白+暖色影 | 2〜3個 |
| 暗い空 | なし or 極薄1個 | 0〜1個 |

### 禁止
- ボクセル/ピクセル風の角ばった雲
- 雲が画面上端で切れる配置

---

## 7. 月・星（暗い空のみ）

### 月

| 項目 | 値 |
|------|-----|
| 形状 | 三日月 / 半月 / 満月 から選択 |
| サイズ | 画面高さの **4〜6%** |
| 位置 | 画面上部エリア。建物と重ならない位置 |
| 色 | 白〜クリーム `#F0E8D8` + 周囲にソフトグロー |
| レンダリング | スムース（ボクセルにしない） |
| 出現条件 | トワイライト〜夜の空のみ |

### 星

| 項目 | 値 |
|------|-----|
| 形状 | 小点(2〜3px) + 十字型(5〜7px)の2種 |
| 色 | 白〜クリーム。透明度60〜100% |
| 密度 | 上部に集中、下にいくほどまばら |
| 個数 | 30〜50個 |
| 出現条件 | マジックアワー〜夜の空 |

## 8. 建物デザイン共通ルール

### 色
- メイン建物は **2〜3色** 構成（ベース60%、サブ30%、アクセント10%）
- ベースとサブは **異なる色相**
- 鮮やかでポップ。くすませない

### 形状
- 現実の建築物をベースにファンタジー化
- 丸み・曲線を積極的に使う
- 窓は丸窓（ポートホール）or アーチ型推奨
- 入口はアーチ型ドア基本

### ライティング
- メイン光源: 正面やや左上
- 窓の光: 暖色 `#FFB060`〜`#FFC870`
- 昼空=明るいフラット、夕焼け=暖色リムライト、夜=暗めアンビエント+窓の暖色光

```
3D fine micro-voxel art building, landscape 16:9, 2K resolution.

[MAIN BUILDING]: [建物の形状・色・ディテール説明]. Built from very small fine voxel blocks (2-4px block size) creating smooth 3D curves with ambient occlusion and smooth shading. The building is centered horizontally, takes up [65-75]% of frame height, with [5-10]% margin from top edge.

[GROUND]: Very thin dark gray ground strip (#4A4A4A) at the bottom, only 3-4% of frame height. Completely flat, no texture, no grass, no decoration.

[SKY]: [空の色・グラデーション説明].

[SIDE BUILDINGS - BARELY VISIBLE, THEME-MATCHED]: Left side: 2-3 tiny [テーマ連動した建物の説明] barely peeking from left edge, only 5-6% of frame width visible, only 15-18% of frame height, almost blending into background. Right side: 2-3 tiny [テーマ連動した建物の説明] barely peeking from right edge, only 3-5% of frame width visible, only 12-15% of frame height. Both sides have STRONG depth of field blur (gaussian 6-10px), very subtle, like background silhouettes with tiny warm window glow. Side buildings are background elements, NOT prominent.

No text, no characters, no grass, no flowers, no ground decoration.
```

## 10. チェックリスト（生成後。全項目パスしないと不合格）

### 構図
- [ ] メイン建物が画面水平中央にある
- [ ] メイン建物の高さが画面高さの60〜75%に収まっている
- [ ] メイン建物の頂点と画面上端の間に5〜10%の余白がある
- [ ] 地面ストリップが画面下部 **3〜4%** にある（8-10%ではない！）
- [ ] 地面はダークグレーのフラット帯で装飾がない

### ボクセル
- [ ] ブロックが細かい（2〜4px四方。チャンキーでない）
- [ ] 曲面がなめらかに表現されている
- [ ] AOが適用されている

### サイド建物（最重要。v4d_castle_bg.pngと照合）
- [ ] 左右に各2〜3棟のサイド建物がある
- [ ] **メイン建物のテーマ・世界観と連動したデザインになっている**
- [ ] 左: 画面幅の **5〜6%** 露出、画面高さの **15〜18%**
- [ ] 右: 画面幅の **3〜5%** 露出、画面高さの **12〜15%**
- [ ] サイド建物がほぼ背景に溶け込んでいる（目立たない）
- [ ] 強めの被写界深度ブラーが適用されている
- [ ] メインの **1/5〜1/4** 以下の高さである
- [ ] v4d_castle_bg.pngと同じサイズ感・控えめさになっている

### 空・天体
- [ ] 空のグラデーションが上→下方向
- [ ] 他の建物と同じ空の色を使い回していない
- [ ] 雲がスムース表示（ボクセルでない）
- [ ] 暗い空の場合: 月と星がある

### 禁止事項
- [ ] 草・花・木・道・石畳がない
- [ ] テキストが入っていない
- [ ] キャラクターが入っていない
- [ ] 地面が3〜4%を超えて厚くない
- [ ] サイド建物がメインと無関係なデザインでない
