## 2. キャラクター設計

### メインキャラ: ぽこ

- **見た目**: 丸い白い生き物。頭にひとつだけ「でこ」（小さな突起）がある。手足は短く丸い。目は大きめのドットアイ
- **性格**: マイペース、好奇心旺盛、集中すると周りが見えなくなる、感覚過敏気味（大きな音でびくっとなる）
- **特徴**: ASD的な特性をモチーフにしているが、「ぽこはそういう生き物」として描く。感情表現はシンプル（顔の変化＋汗マーク・キラキラ等の記号）
- **口癖**: 「ぽこ？」（疑問）「ぽこー！」（喜び）

### サブキャラ1: ぼこ

- **見た目**: ぽこより少し大きい黄色い生き物。頭に「ぼこ」（小さなくぼみ）がある
- **性格**: 元気いっぱい、衝動的に動く、じっとしていられない、失敗してもケロッとしている
- **特徴**: ADHD的な特性をモチーフ。ぽこの親友

### サブキャラ2: もこ（保護者的存在）

- **見た目**: 大きめの水色の生き物。丸い体にエプロン。目は優しい半月型
- **性格**: 心配性だけど見守る派。たまに疲れて溶ける（液体化する）
- **特徴**: 親の「あるある」を体現。完璧じゃない保護者像

### SNSで効く根拠

- **非人間キャラ → 実在親子との誤認リスクゼロ**: 障害をテーマにする以上、特定の親子に見えないことが炎上防止に必須
- **ちいかわ型の「ゆるかわ」→ 心理的ハードルが下がる**: 障害の話題は重くなりがち。可愛いキャラが緩衝材になる
- **キャラグッズ展開可能性**: 将来のマネタイズライン（LINEスタンプ等）
- **シンプルな感情表現 → ミーム化しやすい**: 表情パターンが少ないほどスクショ・保存されやすい

## 2. キャラクター設計

### メインキャラ: りく（小学生男児・架空）

- **見た目**: デフォルメされたイラスト風。丸顔、くせっ毛、大きめの目。服はいつも同じボーダーシャツ（こだわり表現）
- **性格**: 好奇心の塊、空気は読めないが悪気ゼロ、独自ルールがある、電車博士
- **特徴**: ASD＋ADHD混合タイプをモチーフ。「困った子」ではなく「困っている子」として描く
- **重要**: 吹き出しの外に「※りくはフィクションのキャラクターです」と毎回明記

### サブキャラ1: まま（りくの母・架空）

- **見た目**: 疲れ目だけど口角は上がってる。ポニーテール、メガネ、エプロン
- **性格**: 試行錯誤の人。最初は困るが、学んで対応を変えていく成長型
- **特徴**: 読者（親）の投影先。完璧ではないリアルな親像

### サブキャラ2: ふくろう先生（解説役）

- **見た目**: 小さいふくろう。博士帽。吹き出しで科学的解説を担当
- **性格**: 穏やか、押し付けない、「こういう見方もありますよ」スタンス
- **特徴**: 専門知識の出典元キャラ。エビデンスを親しみやすく伝える装置

### SNSで効く根拠

- **デフォルメ人間キャラ → 親近感と距離感の両立**: 完全にリアルだと「うちの子と違う」になる。デフォルメで抽象化
- **解説キャラの分離 → 説教臭さ回避**: 親キャラが教える構図だと上から目線に見える。第三者（ふくろう）が解説することで中立性を保つ
- **成長する親キャラ → 読者の自己効力感UP**: 「この人もできるようになったなら私も」というモデリング効果
- **毎回のフィクション明示 → 炎上リスク管理**: 「実話ですか？」への先回り対策

# ABテスト設計

## 7. KPI・判断基準

### 測定期間
- 各10投稿完了後（約3〜5週間後）に集計

### 主要KPI

| 指標 | 計測方法 | 意味 |
|------|----------|------|
| **保存率** | 保存数 ÷ リーチ数 ×100 | コンテンツの「後で見返したい」価値。最重要指標 |
| **シェア率** | シェア数 ÷ リーチ数 ×100 | 口コミ拡散力。バイラル性の指標 |
| **フォロー転換率** | 新規フォロー数 ÷ プロフィール訪問数 ×100 | アカウントの継続価値 |
| **エンゲージメント率** | (いいね+コメント+保存+シェア) ÷ リーチ数 ×100 | 総合的な反応度 |
| **リーチ数** | Instagram Insights | コンテンツの到達力 |

### 判断基準

| 判定 | 条件 |
|------|------|
| **A圧勝** | Aの保存率・シェア率がBの1.5倍以上 → Aの方向性で本格運用 |
| **B圧勝** | Bの保存率・シェア率がAの1.5倍以上 → Bの方向性で本格運用 |
| **僅差** | 差が1.5倍未満 → 両方継続し、20投稿まで延長テスト |
| **両方低調** | 両方のエンゲージメント率が2%未満 → コンテンツ内容・ハッシュタグ・投稿時間を見直し |

### 副次的に見る指標

| 指標 | 見るポイント |
|------|-------------|
| **投稿別パフォーマンス** | どのネタが跳ねたか。「ネタのジャンル」で分類して傾向分析 |
| **コメント内容** | 共感系（「うちも！」）vs 質問系（「どうすれば？」）の比率 |
| **フォロワー属性** | 年齢・性別・地域がターゲットと一致しているか |
| **プロフィールリンクCTR** | 将来のサービス誘導への導線テスト |
| **最もパフォーマンスが高い曜日・時間** | 投稿スケジュール最適化のデータ |

### 最終判断後のアクション

1. **勝ちパターンを特定** → そのアカウントに投稿頻度を集中（週3〜4に増加）
2. **負けアカウントの扱い** → 即停止ではなく、勝ちパターンの要素を取り入れてピボット検討
3. **勝ちネタのパターン分析** → 「感覚過敏ネタが強い」「対応法つきが保存される」等の法則を抽出
