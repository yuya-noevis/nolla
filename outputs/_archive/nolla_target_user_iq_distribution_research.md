# Nolla対象ユーザー IQ分布リサーチレポート

## 概要

Nollaアプリの対象ユーザー（3-18歳、認知機能に課題のある子ども）のIQ分布を理解するための基礎統計データをまとめたレポート。日本の最新統計、学術論文、およびグローバルデータ（米国CDC）に基づいている。

---

## 2. 各障害のIQ分布

### 2.1 ASD（自閉スペクトラム症）児のIQ分布

| IQ範囲 | 割合 | 備考 | 出典 |
|---|---|---|---|
| **知的障害なし（IQ70以上）** | 約60-70% | 平均的なIQ分布を示す | 複合研究データ |
| **知的障害あり（IQ70以下）** | 約30-40% | 内訳を参照 | CDC ADDM 2023, J-Stage |
| IQ51-70（軽度知的障害） | 約10-15% | 推定 | - |
| IQ36-50（中度知的障害） | 約10-15% | 推定 | - |
| IQ21-35（重度知的障害） | 約5-10% | 推定 | - |
| IQ20以下（最重度知的障害） | 約2-3% | 稀 | - |

**重要な知見**: ASD児のIQ分布は定型発達児とほぼ同じ平均的な分布を示す。高IQ児が多いわけではなく、低IQ児も含まれている。

**出典**:
- [CDC ADDM Community Report 2025](https://www.cdc.gov/autism/media/pdfs/2025/04/ADDM-Community-Report-SY2022.pdf)（米国2022年データ）: 8歳児のASD児の39.6%がIQ70以下

**出典**: [知的障害関連サイト複合](https://h-navi.jp/column/article/73)

### 2.3 ダウン症児のIQ分布

| IQ範囲 | 備考 | 出典 |
|---|---|---|
| **平均 IQ 40-50** | 中度知的障害層が最多 | ミネルバクリニック / 複合研究 |
| IQ 30-70 | 広い分布幅を示す | - |
| **二峰性分布** | IQ 20前後とIQ 40前後に2つのピーク | 鈴木ビネー検査（日本） |
| 性別差 | 女性の方が男性より有意に高い傾向（女性 IQ 42.4 vs 男性 IQ 29.4） | 日本研究 |

**出典**: [ダウン症のIQ分布に関する研究](https://minerva-clinic.or.jp/nipt/column/intelligence-quotient-of-down-syndrome/)

### 2.4 ADHD児のIQ分布

| 所見 | 出典 |
|---|---|
| ADHD児のIQ分布は定型発達児とほぼ同じ | 複合研究 |
| 高IQのADHD児も、低IQのADHD児も存在 | 個人差が大きい |
| IQ測定にはADHD症状の影響がある可能性 | 注意散漫が検査成績を低下させる |

**注記**: ADHD単独では知的障害を伴わない。併存障害（ASD併存、知的障害併存など）がある場合は別。

### 2.5 LD（学習障害）児のIQ分布

| 所見 | 備考 | 出典 |
|---|---|---|
| **IQ70以上が診断条件** | LDの定義上、知的障害を伴わない | 診断基準 |
| 平均 IQ 85-115 | 正常域 | 複合研究 |
| 学習領域は著しく低い | 読み書き計算で大きな遅延 | - |

## 5. Nolla対象ユーザー層の推定IQ分布

### 5.1 ターゲットユーザーの定義

Nollaの対象ユーザー: ASD・軽度〜重度知的障害・ダウン症・LD・ADHD・グレーゾーン（3-18歳）

### 5.2 推定IQ分布（複合）

| IQ範囲 | 推定割合 | 該当する診断 | 発達段階の目安 |
|---|---|---|---|
| **IQ20-35（重度知的障害）** | 3-5% | 重度知的障害、ダウン症(一部)、ASD+重度 | 1-3歳相当の発達 |
| **IQ36-50（中度知的障害）** | 5-8% | 中度知的障害、ダウン症(多数)、ASD+中度 | 3-5歳相当の発達 |
| **IQ51-70（軽度知的障害）** | 10-15% | 軽度知的障害、ダウン症(一部)、ASD+軽度、LD児(一部) | 5-9歳相当の発達 |
| **IQ71-84（境界知能）** | 8-12% | グレーゾーン、軽度ASD、軽度ADHD、LD児(多数) | 9-12歳相当の発達 |
| **IQ85-115（平均〜高平均）** | 60-70% | 知的障害なしのASD、ADHD、LD児 | 年齢相応〜高機能 |
| **IQ116以上（高IQ）** | 2-5% | 高IQ + ASD/ADHD/LD(能力の凸凹が大きい) | 能力の著しい不均等分布 |

**重要な注釈**:
- Nollaユーザーの大多数（60-70%）は知的障害がない（IQ71以上）
- しかしASD/ADHD/LD単独、または軽度知的障害との併存が前提条件
- 重度知的障害層（IQ35以下）はオンラインゲーム利用が難しく、実ユーザーは限定的
- 境界知能層（IQ71-84）は支援がない「隠れた困難層」として重要なターゲット

## 6. 重要な統計的留意点

### 6.1 診断率の地域差・時系列変化

| 指標 | 地域差 / 時系列 | 出典 |
|---|---|---|
| ASD有病率 | 地域で5%超える地域も存在（診断率の地域差） | 日本研究 |
| LD検出率 | 2012年：6.5% → 2022年：8.8%（+35%増加） | 文部科学省 |
| ADHD診断数 | 過去15年で大幅増加（認知度向上） | 複合研究 |

**解釈**: 統計数値は地域や調査時期により大きく異なる。「正確な全国推計」は難しい。

### 6.2 IQ測定の信頼性

| 側面 | 課題 |
|---|---|
| **IQ70境界線の曖昧性** | 軽度知的障害 vs 境界知能の判定が厳密でない |
| **適応機能の考慮不足** | IQだけで診断されることもあり、実際の生活機能と乖離 |
| **ADHD児のIQ低下** | 検査時の集中力障害でIQが過小評価されるリスク |
| **多言語・文化的背景** | 標準化テストのバイアス |

### 6.3 日本とグローバルの有病率比較

| 障害 | 日本 | グローバル（米国CDC） | 差分 | 考察 |
|---|---|---|---|---|
| ASD | 3.22% | 3.2% | ほぼ同じ | グローバルで日本の数値は代表的 |
| ADHD | 5-7% | 5.3-7.1% | ほぼ同じ | 診断基準は同等（DSM-5） |
| 知的障害 | 1.0-1.5% | 1-3% | 幅あり | 定義による差 |

## 7. Nollaビジネスモデルへの含意

### 7.1 対象ユーザー層の規模感

```
日本の3-18歳児童：約1,500万人

├─ ASD + 何らかの困難（グレーゾーン含む）: 19-46万人 (1.3-3.1%)
├─ ADHD（単独 or 併存）: 30-42万人 (2-2.8%)
├─ 知的障害（軽度〜重度）: 6-9万人 (0.4-0.6%)
├─ LD（学習障害）: 27-53万人 (1.8-3.5%)
└─ 境界知能（支援がない層）: 84万人 (5.6%)

推定市場規模: 100万人超（軽度～中度が中心）
```

### 7.2 プロダクト設計のIQ層別対応

| IQ範囲 | 想定ユーザー数 | 推奨UI / ゲーム設計 | 対応優先度 |
|---|---|---|---|
| IQ20-35 | 少数（5-8万人） | 音声 / 簡潔UI / 監督者手段 | 後発対応 |
| IQ36-50 | 中程度（8-12万人） | 絵記号 + 簡潔ルール / 感覚刺激最小化 | 第2フェーズ |
| IQ51-70 | 多数（15-23万人） | ビジュアル豊か / 短期ミッション / 報酬頻度多い | MVP期に重点 |
| IQ71-84 | 最多（12-18万人） | 標準ゲーム設計 / 進度調整可能 / コレクション要素 | MVP期に重点 |
| IQ85以上 | 大多数（90-105万人） | 標準的なカジュアルゲーム設計 | スケール段階 |

### 7.3 マーケット規模の推定（家庭向け ToC）

| セグメント | 推定ユーザー数 | 浸透目標 | 想定ユーザー数 |
|---|---|---|---|
| ASD児（両親が課題認識） | 19-46万人 | 10% | 2-5万人 |
| LD児（学校経由での気づき） | 27-53万人 | 5% | 1-3万人 |
| ADHD児 | 30-42万人 | 5% | 1-2万人 |
| 境界知能児（医学的診断がない） | 84万人 | 2-3% | 2-3万人 |
| **合計** | **150-225万人** | **平均 5-7%** | **6-13万人** |

## 9. まとめ

### Key Findings

1. **Nollaの対象市場は約100万人超** — ASD + LD + ADHD + 知的障害 + グレーゾーン層の合計

2. **IQ分布は極めて多様** — IQ20-130以上の幅広い層が含まれる。単一のゲーム設計では対応不可能

3. **大多数（60-70%）は知的障害がない** — むしろ「学習や行動に困難」な LD/ADHD/軽度ASD が中心

4. **支援を受けていない「隠れた困難層」が巨大** — 境界知能（IQ71-84）約84万人の大多数が医学的診断を受けておらず、支援がない

5. **診断率は地域・時系列で大きく変動** — 統計数値は参考値。実際の市場ニーズは地域調査が必須

6. **ASD + 知的障害の併存は多いが多様** — 同じ「ASD + 知的障害」でも IQ40 と IQ70 では全く異なる対応が必要

### 推奨アクション

- **MVP段階**: IQ51-100 層（軽度知的障害～平均IQ）に絞り、ビジュアル豊か、ルール予測可能、報酬頻度多いゲーム設計
- **スケール段階（Phase 2）**: IQ36-50層向けに音声 + 簡潔UI 版を開発
- **マーケティング**: 「診断がない層」（境界知能、グレーゾーン）への リーチが重要。医者経由ではなく、教育委員会や児童養護施設経由での認知が鍵

## 附録: 用語定義

| 用語 | IQ / 定義 | 備考 |
|---|---|---|
| **最重度知的障害** | IQ ≤ 20 | 完全介護が必要。オンラインゲーム利用は困難 |
| **重度知的障害** | IQ 21-35 | 日常生活に大きな支援が必要 |
| **中度知的障害** | IQ 36-50 | 簡潔な指示なら理解可能 |
| **軽度知的障害** | IQ 51-70 | 学習や社会適応に遅れがあるが、支援で改善 |
| **境界知能** | IQ 71-84 | 診断名がない。支援対象外。隠れた困難層 |
| **グレーゾーン** | 診断基準を部分的に満たす | ASD / ADHD / LD の診断には至らないが、何らかの困難がある層 |
| **適応機能** | 日常生活能力（IQとは別） | IQ が同じでも、自炊・衛生管理・対人関係の能力は個人差が大きい |
