## 2. 機能タクソノミー（機能分類体系）

### カテゴリA: コミュニケーション・言語
**提供サービス数: 10+**

| サブカテゴリ | 典型的なAIアプローチ | 提供サービス例 |
|------------|-------------------|--------------|
| AAC（拡大代替コミュニケーション） | 予測テキスト、画像＋音声合成 | Otsimo, Goally, Spoken, Avaz, Proloquo2Go, SpeakAnyWay |
| 発話・発音矯正 | オンデバイス音声認識＋即時フィードバック | SpeechLP, Sara Speech, Otsimo |

## 4. 市場インサイトとNollaへの示唆

### 4.1 機能の充実度ランキング（カテゴリ数）
1. **QTrobot / Otsimo / CognitiveBotics / Goally** — 5〜6カテゴリ（最も包括的）
2. **Moxie / Floreo / NeuroEngage AI** — 4〜5カテゴリ
3. **Jade ND / Understanding Zoe / Storytailor** — 3カテゴリ
4. **Noora / SpeechLP / Cognoa / Tiimo** — 1〜2カテゴリ（特化型）

### 4.2 市場の空白（ホワイトスペース）
1. **日本語対応サービスの不在**: 上記サービスで日本語に完全対応しているものは皆無
2. **3歳未満への対応**: 多くのサービスが4歳以上を対象。1.5〜3歳の早期介入は未開拓
3. **感覚処理の直接支援**: 最も提供サービス数が少ないカテゴリ（4サービスのみ）
4. **保護者の感情ケア**: ほぼ全サービスが「子どもの課題解決」に焦点。保護者自身のバーンアウト・メンタルケアは未対応
5. **施設（ToB）向け統合ツール**: 個別サービスは多いが、児童発達支援・放デイの業務フロー（個別支援計画・記録・請求）に統合されたものは存在しない
6. **知的障害・ダウン症の特化**: ASD中心のサービスが大多数。知的障害・ダウン症への最適化は少ない
7. **日本の福祉制度との統合**: 児童発達支援・放課後等デイサービスの仕組み（個別支援計画、モニタリング、請求）に統合されたサービスは皆無

### 4.3 AIアプローチの主流パターン
| パターン | 採用率 | 説明 |
|---------|-------|------|
| 適応型難易度調整 | 最多 | 子どものパフォーマンスに応じてAIがコンテンツ難易度を自動調整 |
| 進捗データの自動分析・レポート | 多い | プレイ/使用データをAIが集約し、保護者・専門家にレポート提供 |
| LLMチャットボット | 増加中 | 生成AIによる24時間対応の相談・練習相手 |
| コンピュータビジョン | 中程度 | 表情認識・視線追跡・物体認識 |
| 音声認識・分析 | 中程度 | 発話の正確性評価、音声入力 |
| VR/ロボット環境生成 | 少数 | 安全な練習環境の提供（高コスト） |
