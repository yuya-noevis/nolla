# Nolla 競合・参考企業マスターリスト（2026-04-27）

## 0. 本ファイルの目的

ASD/知的障害領域の企業・プロダクトを過不足なく統合した最新マスター。これまでの全調査（2026-03-29, 04-03, 04-07, 04-12, 04-22, 04-25, 04-26, 04-27）+ factcheck訂正を反映。投資家対応・戦略議論・新規調査の起点に使う。

### 採用基準（ハードゲート）
- ASD または知的障害（境界知能・ダウン症含む）を主要ターゲットに含む
- 一次情報で実体確認できる
- ADHD only / LD only / 定型発達向けは除外（ただし参考枠扱いで一部記載）

### 信頼度評価
- ★★★ 一次情報複数で確認
- ★★☆ 二次情報複数で一致
- ★☆☆ 単一ソース、要追加検証
- 【未確認】 確認できなかった項目を明示

# Z. 構造的所見

## Z-1. 業界全体のRCT脆弱性
ベンチマーク6社全員が独立検証の peer-reviewed RCT を発見できず（Dubuのみ実施で主要評価項目失敗）。Nolla の早期RCT実施が**最大の差別化機会**。

## Z-2. 「累計DL」と「事業規模」は別物
- AutiSpark: 3.6M DL / 年商$14M / bootstrap → **事業規模最大**
- Speech Blubs: 6M DL / 年商$6.5M推定
- Otsimo: 1M DL / 年商$1M / 30K MAU → メディア露出は派手だが事業は小さい

## Z-3. 国内介入レイヤーの空白
ASD治療アプリは国内未承認（ENDEAVORRIDEはADHDのみ）。塩野義の ASD パイプラインも未承認。**Nolla の介入レイヤー参入の窓は依然開いている**。

## Z-4. AAC正面参入はNG
海外11社・国内4強でレッドオーシャン。Nolla は機能として AAC を統合（Mini→Standard）する設計。

## Z-5. 国内大手教育系の不在
ベネッセ/学研/公文/Z会/ジャストシステム等は ASD/ID 特化を投入していない。Nolla 先行参入の窓開く。

## Z-6. 創業者ナラティブの威力
- **Otsimo**: 弟Alper(重度ASD)が3ヶ月で読み書き獲得 → Ashoka/Obama/Fortune獲得
- **Speech Blubs**: 創業者2人が当事者(吃音/発語遅延) → TEDx/Forbesメディア露出
- → **Yuyaの当事者性は事業ブランドの最大武器**になる
