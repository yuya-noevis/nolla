### タイトルカード（枚1）
タイトル：「いつもの道じゃないとダメ」
キャラ（息子＋エイリアン人形）のリアクション絵つき。背景はパステルカラー。

### あるあるパート（枚2〜4）

**枚2:**
いつもの帰り道。息子がエイリアン人形を片手にスキップしてる。母もニコニコ歩いてる。穏やかな夕方の空気。

**枚3:**
母「今日はこっちから帰ろっか！近道見つけたの！」
息子、ピタッと止まる。エイリアン人形をギュッと握りしめて目がまんまる。足が地面に根を張ったように動かない。

**枚4:**
結局いつもの道。息子、またエイリアン人形と一緒にスキップ再開。ご機嫌。
母（小さく笑いながら）「…知ってた」
エイリアン人形もなんか嬉しそうな角度。

### Tipsパート（枚5）
```
【「いつもと同じ」が安心のもと】

いつもの道・いつもの順番・いつもの場所——
変わると固まっちゃうこと、ありませんか？

🧠 なぜそうなる？
・「こうなるはず」の予測で安心を作るタイプの子は
　予測と違う展開にびっくりしやすい
・ワガママではなく「安心の作り方」が独特なだけ

✅ こうするといい感じ
・事前に「今日はこっちの道から帰るよ」と予告する
・地図やルートを絵で見せておく
・「いつもの道＋ちょっと寄り道」で少しずつ広げる

🚫 これは逆効果かも
・突然ルートを変えて「大丈夫だよ！」と押し通す
・「なんでそんなことで怒るの？」と理由を問い詰める

※お子さんによって合う方法は違います
```

## 投稿⑪：「名前を呼んでも振り向かない」

### タイトルカード（枚1）
タイトル：「名前を呼んでも振り向かない」
キャラ（息子が背中を向けて集中してる絵）つき。背景はパステルカラー。

### あるあるパート（枚2〜4）

**枚2:**
リビング。息子が積み木を真剣に積んでる。エイリアン人形は横で「監督」。母、キッチンから「ごはんだよ〜」と呼んでる。息子、微動だにしない。

**枚3:**
3コマ構成（母のエスカレーション）。
①キッチンから呼ぶ母。息子、無反応。
②息子の後ろまで来て呼ぶ母。息子、無反応。積み木が2段増えてる。
③息子の真後ろにしゃがんでる母。息子、まだ無反応。エイリアン人形だけ振り返ってる（ように見える角度）。

**枚4:**
母、息子の肩をぽんぽん。息子、やっと振り向く。何事もなかったような顔。
積み木タワーは立派に完成してる（呼ばれてる間にめちゃくちゃ進んでた）。
母（心の中で）「…呼ぶより、来た方が早い」

### Tipsパート（枚5）
```
【名前を呼んでも反応しない子】

何度呼んでも振り向かない。
無視してるの？聞こえてないの？

　　脳がフィルタリングして遮断してしまう

　この特性の子は「深く集中する代わりに他が入らない」
・聞こえていないのでも無視でもなく、脳の処理方式の違い

✅ こうするといい感じ

※お子さんによって合う方法は違います
```

## 投稿⑫：「シャワーが顔にかかるとパニック」

### タイトルカード（枚1）
タイトル：「シャワーが顔にかかるとパニック」
キャラ（息子が顔を覆ってる絵）つき。背景はパステルカラー。

### あるあるパート（枚2〜4）

**枚2:**
お風呂場。息子がバスチェアに座ってる。エイリアン人形はバスタブの縁の乾いた場所に避難中。母、シャワーヘッドを持って準備OK。穏やかな雰囲気。

**枚3:**
シャワーの水が数滴、息子の顔にかかった。
息子、両手で顔をガード。全身を丸めて「もうダメ」の姿勢。激しく首を横に振ってる。
母、慌ててシャワーを引く。もう片手で乾いたタオルに手を伸ばしてる。

**枚4:**
母、手おけで後頭部からそーっとお湯をかけてる。外科医のような真剣な顔。
息子、顔にタオルを押し当てて片目だけ出してる。まだ警戒中だけど、少し落ち着いた。
エイリアン人形は浴室の一番遠い場所に移動済み。
母（心の中で）「…顔にかけない技術、プロ級になってきた」

### Tipsパート（枚5）
```
【顔に水がかかるのが怖い子】

シャワー・洗顔・プール——
顔に水がかかるとパニックになること、ありませんか？

🧠 なぜそうなる？
・ASD児は触覚の感度が定型発達の数倍高いことがある
・特に顔は三叉神経が集中しており、水の刺激が
　「痛み」に近い強さで脳に伝わってしまう
・さらに顔を覆われる感覚が「窒息の恐怖」と結びつき、
　本能的なパニック反応が起きる
・大げさなのではなく、神経の感度が違う

✅ こうするといい感じ
・手おけで後頭部からゆっくり流す（顔にかけない）
・タオルで顔を覆ってから流す
・「ここまでだよ」と水がかかる範囲を事前に伝える

🚫 これは逆効果かも
・「大丈夫！」と頭からシャワーをかける
・「いつまで怖がるの？」と反応を否定する

※お子さんによって合う方法は違います
```
