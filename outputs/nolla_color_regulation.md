# Nolla カラーレギュレーション（厳守）

**作成日**: 2026-04-04
**全面改訂**: 2026-04-10（MC風→Mario Galaxy Movie風に移行）
**適用範囲**: Nollaアプリ内の全画面・全アセット
**根拠**: ASD児の感覚処理研究（JAMA Psychiatry, PMC Sensory Processing）+ Mario Galaxy Movie分析

---

## 1. カラールールの原則

ASD児の85%が色を高強度で知覚する（JAMA Psychiatry）。ただし色そのものが問題なのではなく、使い方次第。

### 1A. NGなもの

| NG | 理由 |
|----|------|
| 蛍光色（ネオンピンク、ネオングリーン等） | 視覚的攻撃性が高い |
| 点滅・フラッシュ（どの色でも） | 光過敏+てんかんリスク |
| 色だけで意味を伝える | 色覚多様性に対応できない |

### 1B. OKなもの

- 赤、黄、青、緑、紫...**どの色も使ってよい**
- **発光・グロウ表現はOK**（蛍光色とは異なる。拡散光で柔らかく発光）
- 暗い宇宙背景に明るい発光要素を配置するGalaxy的表現は問題なし
- 最終調整時に必要なら微調整する

## 2. 宇宙空間レイヤーカラー（背景構成の基盤）

Galaxy風の色は「暗い宇宙の中に宝石が浮かんでいる」構造。5つのレイヤーで構成。

| レイヤー | 色域 | HEX参考値 | 画面占有率 | 役割 |
|---------|------|-----------|-----------|------|
| **深宇宙ベース** | ダークネイビー〜インディゴ | #0A0A2E → #1A1A4E | 60-70% | 安定感の土台 |
| **ネビュラ中間層** | ティール / マゼンタ / ディープパープル | #1B6B6B / #8B2A6B / #3A1A6B | 20-25% | グラデーションで混ざり合う |
| **発光アクセント** | シアン / ゴールド / ライムグリーン | #00D4FF / #FFD700 / #7BFF3A | 5-10% | 星、パーティクル、リムライト |
| **暖色スポット** | アンバー / ウォームイエロー | #FFB347 / #FFF4C1 | 3-5% | 安心感の源 |
| **草・惑星の緑** | ライムグリーン〜エメラルド | #5DBB3D / #3A8F3A | 惑星表面 | 生命感の象徴 |

**色の遷移ルール**: 隣り合う色は必ずグラデーションでなめらかに接続。ハードエッジの色境界は存在しない。ネビュラ層は最低3色が同時に溶け合っている。

各惑星は固有のカラースキームを持つ。ホーム画面のネビュラ色と惑星表面色が対応。

| 惑星 | ゲーム | 球体表面色 | ネビュラ主色 | アクセント光 | アトモスフィア |
|------|--------|-----------|-------------|-------------|-------------|
| カード星 | 神経衰弱 | #9B7BBD（ラベンダー）/ #7B5A9D（影） | ディープパープル→マゼンタ | #DAA520 → #FFD700 ゴールド結晶 | パープルピンク |
| いろわけ星 | 分類 | パッチワーク: #FF8C42 / #7BFF3A / #5BB8F0 / #FF6B8A | ティール→エメラルド | #FFB347 アンバー境界光 | ウォームオレンジ |

## 5. フィードバックカラー

| 状態 | 表現方法 | 色 | 注意 |
|------|---------|-----|------|
| 正解 | チェックマーク + ゴールドパーティクル爆発 | #5DBB5D グリーン + #FFD700 ゴールド | 6-8個のスターパーティクル |
| 不正解 | 正解要素が柔らかく光る | テーマ色のグロー (30% opacity) | エラー表示しない |
| 報酬獲得 | スター飛び出し + ゴールドラジアルグロウ | #FFD700 | 小面積・短時間 |
| 休憩提案 | 宇宙空間がゆっくり暗くなる + 月 | #1A1A4E → #0A0A2E | 穏やかに |
| 選択中 | 柔らかいパルスグロウ | テーマ色の明度+10% | ゆっくり1.5秒周期 |
| ワープ遷移 | 黒背景 + 星が放射状に高速で流れる | #FFFFFF 星 + #00D4FF シアン軌跡 | 1.2秒 |

## 6. エラーレス学習における色の使い方

| 場面 | ルール |
|------|-------|
| 不正解時 | ネガティブ表示をしない。正解要素がテーマ色のグロウで柔らかく光る |
| 正解時 | チェックマーク + ゴールドパーティクル + 音。発光表現自由 |
| 警告・注意（親向け） | アンバー系。赤は使わない。形・アイコンと併用 |

---

## 7. アクセシビリティ補足

- **色覚多様性**: 色だけで意味を伝えない。必ず形状（丸/四角/三角）と組み合わせる
- **コントラスト比**: 親向けテキストは WCAG AA（4.5:1以上）を満たす
- **子ども向け画面**: テキスト不使用のためコントラスト比は適用外。ただしアイコンの視認性は確保
- **暗い背景上のUI**: UIアイコンは必ず半透明ダーク背景パネルの上に配置して視認性確保
- **発光 vs 蛍光**: 発光（グロウ/拡散光/リムライト）はOK。蛍光色（ネオン原色そのもの）はNG

## 8. 色チェックリスト（デザイン/開発時）

全画面に適用。チェックが全て通らないとリリース不可。

- [ ] 蛍光色を使っていないか（グロウ/発光はOK）
- [ ] 点滅・フラッシュする要素がないか
- [ ] 色だけで情報を伝えていないか（形・アイコンと併用しているか）
- [ ] 親向け画面のテキストコントラスト比が4.5:1以上か
- [ ] エラー表示にネガティブ表現を使っていないか
- [ ] 宇宙背景の最暗値が#0A0A2Eより暗くないか
- [ ] ゲーム中の宇宙背景がホーム画面より控えめ（彩度-30%）になっているか
- [ ] 暖色の光源（ゴールド/アンバー）が画面内に最低1つ存在するか
- [ ] 色の遷移がグラデーションで滑らかか（ハードエッジなし）
