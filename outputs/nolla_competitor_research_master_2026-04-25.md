## 3. Nollaへの戦略的含意（更新版）

### 3-1. 「3大ベンチマーク + 副次脆弱性」のフレームは部分的に修正必要

**前回の主張**: 3社合計180万ユーザー・査読RCT実証ゼロ → Nollaの差別化チャンス

**今回の修正**:
- 査読RCT実証ゼロという構造的脆弱性は **業界全体に拡大して当てはまる**（Speech Blubs・AutiSpark・Smart Tales・Genial Care他、ほぼ全社が該当）
- ただし**ユーザー規模では3社の上にSpeech Blubs/AutiSparkがいる**ため、「3社が世界の主導格」というポジショニング主張は**事実誤認**になる
- 投資家ピッチで「3大競合」と言うと、リサーチ甘さを突かれるリスクがある

### 3-2. Nollaの独自ポジションは依然存続するが、語り方を変える必要

**Nollaが市場で空白を埋めるポジション**（変更なし）:
- IQ21-110（知的障害+境界知能）に完全特化
- NCI型3軸スコアリング（実行機能/視覚空間/ワーキングメモリ）の長期累積
- エラーレス学習（成功率75-80%）の自動実装
- ASD児がハマるゲーム性（Minecraft/どうぶつの森型）

**ただし主張の仕方を変える**:
- 「3大競合より優位」→「世界規模で見ても、知的障害特化×ゲーム×個別測定×日本語深化は空白」
- 「査読RCT先取り」→「世界中で誰もRCTで証明できていない領域に、日本市場で先取り挑戦」（業界共通課題への挑戦と位置づけ）

### 3-3. 「Yuya案＝課題逆算型ロードマップ」の差別化はむしろ強化される

世界市場を見渡しても、Yuya案の核心要素である**「親の困りごと（発語・問題行動・着替え等）を起点に、サブタイプ判別 → 認知土台の逆算ロードマップ → コンテンツ配信」を統合しているプレーヤーは見当たらない**:

- Speech Blubs: 発語に特化、認知土台への逆算なし
- AutiSpark: 認知ゲーム集合体、課題起点ではない
- Smart Tales: ストーリーベース学習、課題起点ではない
- Genial Care: サービス統合型（療法士派遣）、ロードマップは人ベース
- Mightier: HRV-基盤、特定領域

**結論**: Yuya案の方向性は、世界規模で見ても空白を突いている。これをコア差別化として明示すべき。

### 3-4. AAC市場には参入しない判断は維持

- Proloquo2Go / TouchChat / LAMP / Avaz / Leeloo / CoughDrop / Niki Talk / JABtalk / Spoken / GoTalk Now / Predictable の **11社が累計数百万ASD/IDユーザー保有**
- NollaがAACに正面参入すると12社目になるレッドオーシャン
- **AAC Mini的な軽機能**を持ちつつ、コアは認知トレーニング+ロードマップに集中する戦略が正解

### 3-5. B2G/政府連携は世界トレンド、日本でも先取りすべき

世界の成功モデルが集中しているチャネル:
- **Dubu** → 韓国政府・8,000+特支学級
- **BlinkLab** → モロッコ政府・年60万児
- **Auticiel** → 仏MDPH助成連動
- **Cogs AI / XR Therapeutics** → UK NHS連携
- **Compedia/EmotiPlay** → Cambridge ARC学術連携

→ **Nollaも放課後デイ・特支学校・自治体教育委員会・国立成育医療研究センター等への先行アプローチが必須**

### 3-6. 中国圏は日本未進出のため脅威度低、ただし将来的にウォッチ

- Dami & Xiaomi（$62M調達）/ ALSOLIFE（27万家庭）の中国2強は日本進出予定なし
- ただし両社ともグローバル展開意欲あり（Tencent SSV経由でのASEAN展開示唆）
- 2-3年後の日本展開リスクを念頭に、**ASEAN市場の先行確保**を視野に入れるべき

## 6. 結論

前回の「Dubuを見落としていた」という指摘は正しかったが、**今回の徹底調査で見えたのは、Dubu以外にも見落としていた巨大プレーヤーが少なくとも12社以上存在した**ということ。

ただし**Nollaの戦略の根幹（IQ21-110特化×3軸NCI×エラーレス学習×ハマるゲーム×日本語深化）は依然として独自**。Yuya案の課題逆算型ロードマップは世界規模で見ても空白を突く方向性として有効。

修正すべきは**「世界の主導格は3社しかいない」「我々が業界初」のような言い回し**。これは事実誤認になり、投資家・パートナーから即突かれる。代わりに「世界の支援テック市場全体に共通する構造的脆弱性に、日本市場の特異性を武器に挑む」という枠組みが正しい。
