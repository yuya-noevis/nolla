# Nolla — Mario Galaxy Style Regulation

## 使い方

1. 変換したい素材画像をAI画像生成ツールに添付する
2. 下記の **Master Prompt** をそのままコピペして送信する
3. 惑星ごとに色調整が必要な場合は **個別カラーマップ（セクション3）** の該当部分を追記する

それだけ。

---

## Master Prompt

```
添付画像を以下のルールで変換しろ。構図・モチーフ・オブジェクトは一切変更するな。色彩・質感・光だけを変える。

【絶対禁止】
- 花びら、葉、羽根、パーティクル、星屑、ボケ要素など、元画像にないものを一切追加するな
- 構図を変えるな、モチーフを足すな引くな
- アドリブで「良かれと思って」何かを付け足すな。指示にないことはするな

【質感の変換】
- フォトリアルなCGテクスチャ → 水彩/ガッシュで描いたようなペインタリーな質感に変換
- シャープなCGエッジ → 輪郭がわずかに背景に滲むソフトエッジに変換
- PBRの金属光沢/鏡面反射 → 内側から柔らかく光る拡散反射に変換
- デジタルのクリーンなグラデーション → 水彩のような微かなムラ・にじみのあるグラデーションに変換
- 画像全体に水彩紙(Cold-pressed)の微かな紙目テクスチャを5-8%の強さで重ねる

【ライティングの変換】
- ハードな直接光/ポイントライト → 左上からの柔らかいディフューズドライト(拡散光)に変換
- 暗い影/真っ暗な影領域 → 暖色寄りの透明な影に変換。最暗部でも色味が残る(#1A2A4A程度)
- 高コントラスト → 低コントラスト/ハイキーに。暗部を全体的に持ち上げる
- 惑星の輪郭に白〜クリーム(#FFF8E8)のリムライトを追加。幅はオブジェクトの2-3%、不透明度50-70%
- 惑星の球面エッジにSSS(サブサーフェス・スキャタリング)。エッジ付近が明るく、内側に自然減衰

【色調整】
- 色相は変えない。明度と彩度だけ調整する
- 暗い影部分: 明度+20〜30%引き上げ
- ハイライト部分: 彩度-10%、明度+10%。クリーミーな光に
- 中間トーン: わずかに暖色側にシフト(+200K程度)
- 発光要素(窓の光など): 色はそのまま、エッジをソフトに(にじませる)

【最終確認】
- 元画像にないものが追加されていないか確認しろ
- 構図が変わっていないか確認しろ
- フォトリアルな質感が残っていないか確認しろ
```

## 個別カラーマップ（必要な場合のみMaster Promptの末尾に追記）

### ひかり星（Corsi Block / 夜の街並み惑星）

```
【追加色指定】
- 建物のダークブルー #1A1A3A → ミッドインディゴ #4A5A7B
- 道路/石畳 #1A1A2A → ウォームグレー #3A3A4B
- 窓の暖色光: 色はそのまま維持、グロー(にじみ)を強める
- 最暗部: #1A2A4A より暗くしない
```

### カード星（Memory Match / 紫ベルベット惑星）

```
【追加色指定】
- ダークパープル表面 #3A2A4A → ソフトラベンダー #6B5A7B → #8B7A9B
- クリスタル: 透明度を上げ、内部に光の透過感を出す
- ゴールド要素 #FFD700 → クリームゴールド #F0E0A0
- 苔/緑部分: 明度+10%のみ
```

### いろわけ星（Sorting / 4バイオーム惑星）

```
【追加色指定】
- 砂漠セクション: 変更不要(既に暖色で明るい)
- 海セクション #2A7AAB → #5A9AC8(明度上げ、透明感)
- 森セクション: 変更不要(緑は既にGalaxy的)
- 溶岩セクション #8B2A00 → #AB5A30(暗い赤を暖かいオレンジ寄りに)
- バイオーム間の境界: やや滑らかに
```

### さがし星（Visual Search / 緑の草原惑星）

```
【追加色指定】
- 変換量は最小。最もGalaxy Styleに近い素材
- 草の緑: そのまま維持
- 池の青: 明度+10%
- 家の屋根: そのまま維持
- ハイライト部分にクリーム感をわずかに追加するのみ
```

### 宇宙背景（背景.jpg）

```
【追加色指定】
- 既にセルリアンブルー系。大きな変更不要
- 既存のラベンダー星雲: 明度+20%のみ
- 既存の星: クリームゴールド(#FFF4D0)寄りに微調整
- 既存の星雲テクスチャ: そのまま残す
- 質感をペインタリーに変換するのみ
```

---

## 変換の核心（3行で）

1. **フォトリアル → ペインタリー**（CGレンダーではなく水彩画）
2. **暗い影 → 暖色の透明な影**（真っ暗な領域は存在しない）
3. **何も足さない**（質感・色・光の変換のみ。装飾追加は一切禁止）
