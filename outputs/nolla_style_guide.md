## 2. キャラクターデザイン

### ノラ（メインキャラ）

| 項目 | 仕様 |
|------|------|
| 種族 | 小さな丸い生き物（種族不明。人でも動物でもない） |
| 体型 | 2頭身。まんまるの頭+小さなからだ |
| 色 | からだ: クリームイエロー(#FFF3D4) / ほっぺ: うすピンク(#FFD4CC) |
| 帽子 | やわらかいブルー(#6BAED6)のまるい帽子。常に着用 |
| 目 | 大きくまんまる。黒目がち。キラキラのハイライト1つ |
| 口 | シンプルな曲線。5パターン：にこにこ / 大きく開ける / まる / むー / びっくり |
| 手足 | 短くて丸い。指は描かない（ミトン状） |
| 輪郭線 | やわらかいブラウン(#C4A882)。太さ均一。黒線は使わない |
| 動き | ぴょんぴょん跳ねる / 手を振る / 首をかしげる。急な動きはしない |

### サブキャラクター共通ルール

| 項目 | 仕様 |
|------|------|
| 頭身 | 2頭身（ノラと統一） |
| 輪郭線 | ブラウン(#C4A882)。ノラと同じ |
| 表情 | ノラと同じ5パターン |
| 色数 | 1キャラ3色以内（体色 + アクセント1色 + ほっぺ） |
| デフォルメ | 丸みを強調。角ばった要素なし |

### NOLLA-001 登場キャラ

| キャラ | 体色 | 特徴 | アクセント色 |
|--------|------|------|-------------|
| おひさま | あたたかいオレンジ(#F5A623) | 丸い。光の線が4本（シンプル）。常にニコニコ | うすイエロー(#FFF3D4) |
| うさぎ | うすピンク(#FFD4CC) | 長い耳。丸い体。眠そう→元気になる変化 | 白(#FFF8F0)のおなか |
| くま | ナチュラルブラウン(#C4A882) | 丸い耳。大きめの体。おっとり | クリームイエロー(#FFF3D4)のおなか |

## 3. 背景デザイン

### 基本ルール

| 項目 | 仕様 |
|------|------|
| 複雑さ | **極めてシンプル**。地面の線 + 空のグラデーション程度 |
| 情報量 | キャラ + 背景要素1-2個まで（雲、木、花など） |
| テクスチャ | フラット。グラデーションは穏やかに |
| パターン | 縞模様・チェック・水玉 **禁止** |
| 遠近感 | なし or 最小限。平面的な構成 |

### NOLLA-001 シーン別背景

| シーン | 背景 | 色 |
|--------|------|-----|
| オープニング/エンディング | クリームホワイト無地 | #FFF8F0 |
| テーマ提示（朝） | 下から上に うすピンク→そらいろ のグラデーション | #FFD4CC → #B3DAF1 |
| うさぎの場面 | わかくさの丘 + そらいろの空 | #C8E6C9 + #B3DAF1 |
| くまの場面 | 同上 + 小さな木1本 | 同上 + #C4A882 |
| 振り返り | 明るい朝の光。クリームホワイト + おひさまの光（ゆるいグラデーション） | #FFF8F0 + #FFF3D4 |

## 4. テロップ・テキスト

| 項目 | 仕様 |
|------|------|
| フォント | 丸ゴシック系（例: M PLUS Rounded 1c, ヒラギノ丸ゴ） |
| サイズ | 画面幅の60%以上 |
| 色 | ダークブラウン(#4A4A4A) on クリームホワイト(#FFF8F0) |
| 縁取り | 白(#FFFFFF) 2px。読みやすさ確保 |
| 表示時間 | 読む速度の1.5倍以上 |
| 同時表示 | 1テロップまで |
| インバーバル空欄 | 空欄部分にキラキラエフェクト（穏やか） |

## 5. アニメーション・動き

### 許可される動き
| 動き | 速度 | 使用場面 |
|------|------|---------|
| ぴょんと登場 | 0.5秒 ease-out | キャラ登場 |
| 手振り | ゆっくり2往復 | あいさつ |
| ジャンプ（喜び） | 0.3秒 ease-in-out | 強化ポイント |
| 首かしげ | 0.5秒 ease-in-out | 問いかけ |
| クロスフェード | 0.3-0.5秒 | 場面転換 |
| ゆっくり浮上 | 1-2秒 linear | おひさまの登場 |

### 禁止される動き
- フラッシュ・点滅（一切不可）
- 画面全体の急激な色変化
- 回転系エフェクト
- ズームイン/アウトの高速切り替え
- 振動・シェイクエフェクト

## 6. 画面レイアウト

### 基本レイアウト（縦型 9:16）

```
┌──────────────┐
│              │
│   [おひさま]  │  ← 上部1/4: 空・環境要素
│              │
│              │
│   [キャラ]    │  ← 中央: メインキャラ（画面の40-50%）
│              │
│              │
│  [テロップ]   │  ← 下部1/4: テロップ（ある場合）
│              │
└──────────────┘
```

- キャラは常に画面中央〜やや下に配置
- 目線の高さが画面の中央になるように
- 余白を十分に取る（窮屈にしない）

## 7. サムネイルテンプレート

### 構成

```
┌──────────────────┐
│ ★               │  ← 左上: レベル星マーク
│                  │
│    [ノラの顔]     │  ← 中央: キャラクターの顔（大きく）
│                  │
│   「おはよう」    │  ← 下部: テーマ文字（3-5文字）
│                  │
└──────────────────┘
背景: レベル色で塗りつぶし
```

| 要素 | 仕様 |
|------|------|
| 背景色 | レベル別色（NOLLA-001は #A8D8B9） |
| キャラ | 顔のアップ。笑顔 |
| テキスト | テーマを1-2語。丸ゴシック、白縁取り |
| 星マーク | 左上。レベルに応じた数 |
| サイズ | 1280 x 720 px |
