# Nolla Phase 1 数値目標 + ユーザー獲得戦略 v1

## 0. 本書の位置づけ

設計書再評価で「Phase 1までの数値目標(DAU/継続/転換率)が未明記」と指摘された(-1点)。
本書はPhase 0完了→Phase 1(MVP-Alpha公開)の3-6ヶ月間で達成すべき定量目標と、その根拠・獲得チャネル設計をまとめる。

**前提**: 
- ターゲット = ASD/知的障害/ADHD/LD/境界知能児の親(国内市場まず)
- 配布チャネル = PWA(Webアクセス)+ App Store / Google Play(将来)
- 価格 = MVP無料、Phase 1で課金実験開始

## 2. KPI別詳細

### 2.1 獲得 (Acquisition)

**目標: 1,000家庭サインアップ**

| ファネル | 目標数 | 想定転換率 |
|---|---|---|
| 認知(LP訪問) | 50,000 | 100% |
| 興味(サインアップページ到達) | 5,000 | 10% |
| 試行(サインアップ完了) | 1,500 | 30% |
| 起動(初回ログイン) | 1,000 | 67% |

**サインアップ数の月次推移目標**:
- 1ヶ月目: 100
- 2ヶ月目: 100
- 3ヶ月目: 150
- 4ヶ月目: 200
- 5ヶ月目: 200
- 6ヶ月目: 250

### 2.2 アクティベーション (Activation)

**目標: 初回プレイ完了率 75%**

「初回プレイ完了」=オンボーディング(S1〜S6)→ホーム画面到達→1ゲーム1ラウンド完了。

| ステップ | 目標到達率 | リスク |
|---|---|---|
| S1 スプラッシュ突破 | 100% | (自動遷移) |
| S2 アカウント作成 | 90% | OAuth失敗、メール認証離脱 |
| S3 アセスメント完了 | 85% | 質問が長すぎ |
| S4 同意取得 | 95% | プライバシーポリシー長すぎ離脱 |
| S5 端末引き渡し | 95% | 親が子に渡せず |
| S6 音声ON/OFF | 95% | 設定迷い |
| ホーム画面到達 | 90% | チュートリアル離脱 |
| 1ゲーム1ラウンド完了 | 85% | 操作習熟、難度ミスマッチ |
| **合計(累積)** | **75%** | |

### 2.3 エンゲージメント (Engagement)

**目標: DAU(平日)200、WAU/DAU 4.0**

- 平日: 子ども帰宅後の17-21時にピーク予想
- 週末: 全日プレイ可能性

DAUの内訳予想:
- 毎日プレイ層: 80家庭(40%)
- 週3-4回層: 80家庭(40%)
- 週1-2回層: 40家庭(20%)

セッション時間目標: 1セッション 8-15分(集中持続時間の上限に合わせる)

### 2.4 継続率 (Retention)

**目標: D7=50% / D30=30% / D90=15%**

業界比較:
- 一般子ども向け教育アプリ D7=40% / D30=20% / D90=8%
- 高品質ゲームアプリ D7=60% / D30=30% / D90=15%

NollaはASD児の高ロイヤリティ(予測可能性が好まれる)を活かして業界平均を上回る目標。

リテンション施策:
- 適応型難度で「やりすぎてつまらない」「難しすぎて挫折」を防ぐ
- 報酬ショップ+My Roomで「次もやりたい」モチベーション
- スター永続化+図鑑コンプリート要素
- 親通知(週次NCIレポート)で親側継続意欲

### 2.5 満足度 (Satisfaction)

**目標: NPS 40+(親調査)**

調査タイミング:
- D7時点でアプリ内アンケート
- D30時点でメール調査

質問: 「Nollaを発達障害のお子さんを持つ友人に勧める可能性はどれくらいですか?(0-10)」

NPS = 推奨者(9-10)% - 批判者(0-6)%

参考:
- Toca Boca, Khan Academy Kidsクラス: NPS 60-70
- 一般教育SaaS: 30-40
- Nolla目標 40 = SaaS平均上、ベンチマークアプリ近づく中間水準

### 2.6 データ品質 (Data Quality)

**目標: NCI測定可能率 70%**

「測定可能」=ベースライン確立(2週間程度のプレイデータでSE±15以下に収束)。

未達リスク:
- プレイ頻度不足(週3未満)
- 異常検知発火(乱暴プレイ・親代理プレイ)
- ゲーム種類偏り(4ゲーム×2軸ずつカバーが必要)

施策:
- 親通知「もう少しプレイすると正確な記録ができます」
- 4ゲームを均等に推奨するアルゴリズム
- 異常検知ログを親ダッシュボードで提示しない(子のモチベ維持)

### 2.7 臨床価値 (Clinical Value)

**目標: NCI 30日で +3以上 = 50%家庭**

NCI 1ポイント = WISC-V換算で約0.3標準偏差(仮説)。
30日で +3 = 約1標準偏差の改善 = 統計的に有意な変化。

これが達成できれば、親の「効果実感」が高まり、NPS・継続率・有料転換に好循環。

未達時の対応:
- ベースライン期間を3週間に延長(1週間→効果が見える)
- ゲームメカニクスの再調整
- 過剰配慮(設計境界の見直し)

### 2.8 収益 (Revenue)

**目標: 課金転換率 10%(課金導入後)**

Phase 1前半は無料運用、後半(4-6ヶ月目)で課金実験開始予定。

価格仮説:
- フリープラン: 1日10分まで、4ゲーム使い放題、NCIスコア表示
- プレミアム: ¥980/月、無制限プレイ、AI Agent、詳細レポート、図鑑全開放

転換率の業界比較:
- フリーミアムSaaS平均: 2-5%
- 子ども向け教育アプリ: 5-10%
- Nolla目標 10% = 業界上位

## 3. ユーザー獲得チャネル設計

### 3.1 チャネル一覧と優先度

| チャネル | コスト | 想定獲得数(6ヶ月) | 優先度 |
|---|---|---|---|
| **わが家SNS連携** | 低(自社SNS活用) | 400 | ★★★ |
| **医師・療育士推薦** | 中(リレーション構築) | 200 | ★★★ |
| **発達支援センター・特別支援学校** | 中(B2G2C) | 200 | ★★ |
| **保護者コミュニティ・LINE** | 低(オーガニック) | 100 | ★★ |
| **Google検索 SEO** | 低(コンテンツ作成) | 50 | ★ |
| **広告(Meta/Google)** | 高(CAC高) | 50 | ★ |

### 3.2 チャネル別具体施策

#### A. わが家SNS連携(目標 400獲得)

**現状**:
- わが家アカウント(Instagram/X)が運用中
- 「今日もわが家は通常運転」発信中
- 既にフォロワー基盤あり

**施策**:
- わが家コンテンツ内にNollaの自然な紹介(月2回)
- わが家キャラ→Nolla内キャラとの連動(報酬ショップで配布)
- フォロワー限定ベータ招待(早期100名)
- インフルエンサー協業(同じカテゴリの親アカウント5名)

**KPI**: わが家フォロワー → Nolla 転換率 5%

#### B. 医師・療育士推薦(目標 200獲得)

**ターゲット**:
- 小児神経科医、発達外来医
- 言語聴覚士(ST)、作業療法士(OT)
- 児童発達支援事業所スタッフ

**アプローチ**:
- Phase 0: ターゲット医師50名リストアップ
- Phase 1前半: 個別アプローチ(LinkedIn/学会)で20名にデモ
- 推薦インセンティブ:
  - 患者紹介料(月¥5,000-10,000)
  - 臨床データへの早期アクセス権
  - 学会発表での共著権利

**KPI**: 1医師あたり10家庭紹介

#### C. 発達支援センター・特別支援学校(目標 200獲得)

**B2G2C モデル**:
- 自治体経由で公立特別支援学校にアプローチ
- 民間事業所経由で家庭にリーチ

**Phase 1施策**:
- パイロット校5校確保
- 学校側ダッシュボード(クラス全体NCI分布)を提供
- 教員研修会の実施(月1回)

**KPI**: 1校あたり40家庭利用

#### D. 保護者コミュニティ(目標 100獲得)

- LINEオープンチャット「ASD親の会」等にアプローチ
- ピアレビュー型のクチコミ形成
- ベータ招待→体験談シェアのインセンティブ(プレミアム1ヶ月無料)

#### E. SEO(目標 50獲得)

ブログコンテンツ:
- 「ASD児の認知トレーニング 自宅でできる方法」
- 「IQ50の子のための学習アプリ」
- 「言葉が出ない子のコミュニケーション支援ツール」

検索ボリューム調査済みキーワード10本×3ヶ月で記事化。

#### F. 広告(目標 50獲得)

CAC試算:
- Meta広告 CPC ¥100 × CVR 3% = CPA ¥3,300
- 6ヶ月で50獲得 = ¥165,000
- LTV(初年度): ¥980 × 12ヶ月 × 10%課金 = ¥1,176/家庭(全体平均)
- LTV/CAC = 0.36 → 広告は赤字

**判断**: 広告はオーガニック成長の補完のみ。メインチャネルにしない。

## 4. 計測体制

### 4.1 必要な計測ツール

| 用途 | ツール | コスト |
|---|---|---|
| プロダクト分析 | PostHog (OSSセルフホスト) or Mixpanel | 無料-¥10k/月 |
| エラー監視 | Sentry | 無料-¥5k/月 |
| Webサイト解析 | Plausible (児童データ非送信) | ¥1k/月 |
| NPS調査 | Typeform + メール送信 | 無料-¥5k/月 |
| ファネル分析 | Supabase Studio + 自作SQL | 無料 |

**注意**: COPPA/GDPR上、Google Analyticsは児童データに使えない。代替を選定。

### 4.2 主要ダッシュボード

1. **獲得ファネル**: LP訪問→サインアップ→初回プレイ
2. **エンゲージメント**: DAU/WAU/MAU推移、セッション時間分布
3. **継続率**: D1/D7/D30/D90コホート
4. **NCIヘルス**: 測定可能率、ベースライン到達日数分布
5. **満足度**: NPS推移、アンケート自由記述

### 4.3 週次レビュー項目

- 新規サインアップ数とチャネル別内訳
- DAU推移とMA(7日)
- 直近1週間のリテンション
- エラー率・クラッシュ率
- ハイリスク警告の発火数(AI Agent)
- 親からのフィードバック(Top 3課題)

## 5. リスクと対応

| リスク | 影響 | 対応 |
|---|---|---|
| 医師アプローチが想定より時間かかる | 200家庭達成不可 | わが家SNSの比率を上げる |
| わが家SNSのエンゲージメントが伸びない | 400家庭達成不可 | コンテンツ戦略の見直し |
| 学校パイロットが許可されない | 200家庭達成不可 | 民間事業所経由に切替 |
| 課金転換率が5%以下 | 収益モデル成立せず | 価格・特典の再設計 |
| NCI測定可能率が50%以下 | 効果立証困難 | ベースライン期間延長、ゲーム調整 |

## 6. Phase 2への移行条件

Phase 1終了時(6ヶ月後)に以下を達成すれば Phase 2へ:

- [ ] 累計サインアップ 500以上(目標1000の50%)
- [ ] D30継続率 20%以上(目標30%の67%)
- [ ] NPS 30以上(目標40の75%)
- [ ] NCI測定可能率 50%以上(目標70%の71%)
- [ ] 課金転換率 5%以上(目標10%の50%)

最低基準を下回る場合は Phase 1延長、メカニクス再設計、または事業ピボット判断。
