1. 最も効果が高い認知機能TOP3
================================================================================

【第1位】実行機能 / 認知的柔軟性 (Executive Function / Cognitive Flexibility)
──────────────────────────────────────────────────────────────────────
エビデンスレベル: ★★★★★
Effect Size: 中～大（g > 0.50）
推奨ゲーム形式: ルール切り替え型（Predictable rule shift）
セッション: 15分 × 2-3回/週 × 18週間
期待改善期間: 4-8週

キー研究:
  - メタ分析（59研究, n=2,122）: 認知的柔軟性で小～中程度の改善
  - デジタルモーション感知ゲーム: 18週で有意改善（最も強い効果）
  - Braingame Brian RCT: Executive function改善のトレンド認証

禁止事項:
  ✗ 時間制限（時間圧力でパニック）
  ✗ ルール突然変更（予測不可能性）
  ✗ 複雑なビジュアル（感覚overload）

実装ポイント:
  ✓ ルール明示的変更（音+画面表示）
  ✓ Errorless learning（サイレント修正）
  ✓ Variable Ratio強化スケジュール (VR3)
  ✓ Success rate 75-80%を自動調整


【第2位】ワーキングメモリ (Working Memory)
──────────────────────────────────────────────────────────────────────
エビデンスレベル: ★★★★
Effect Size: 中程度（g=0.63-0.64）
推奨ゲーム形式: マッチング + シーケンス記憶
セッション: 20-30分 × 5-6週間（集中的）
期待改善期間: 4週以内に観察可能

キー研究:
  - CogMed WMT（ASD+ID, n=26）: 96%完了、>88%満足度
  - Jungle Memory: Effect size g=0.63-0.64（言語的WM）
  - Nature Scientific Reports: スマートフォンゲームで有効検証

実装ポイント:
  ✓ CRF初期 → VR3-5（習得後）
  ✓ 即座フィードバック（< 200ms）
  ✓ Fuzzy logic adaptive difficulty
  ✓ SR 75-80%を自動調整


【第3位】処理速度 + 注意制御 (Processing Speed + Attention)
──────────────────────────────────────────────────────────────────────
エビデンスレベル: ★★★
Effect Size: 小～中（継続的介入が必須）
推奨ゲーム形式: アクションゲーム / 注意訓練
セッション: 継続的（介入時間が長いほど効果的）
期待改善期間: 8-12週

キー研究:
  - CPAT（Computerized Progressive Attentional Training）: 認知・学業成績改善
  - アクションゲーム: 処理速度・選択的注意・視空間推論に正の影響
  - 9/11研究: 読解力・数学・反応時間の改善報告

制約:
  ✗ 処理速度改善は他の認知機能より緩やか
  ✗ 時間制限なし（anxiety軽減）

3. ABA強化スケジュール推奨
================================================================================

【推奨】Variable Ratio (VR) スケジュール
───────────────────────────────────────

初期段階（習得フェーズ）:
  • Continuous Reinforcement (CRF)
  • 全正解に即座報酬（< 200ms）

習得後（維持フェーズ）:
  • Variable Ratio 3-5 (VR3-VR5)
  • 平均3-5正解ごとに報酬
  • 予測不可能 → 継続的エンゲージメント

理由:
  • Fixed Ratio: 報酬後に一時停止が生じやすい
  • Variable Ratio: 一定の反応率を維持（ゲーム的）

4. ASD+ID児向けゲーム設計の禁忌事項
================================================================================

【禁止1】時間制限の厳格な回避
──────────────────────────────────

❌ Countdown timer（視覚的時間圧力）
❌ 「急いで！」メッセージ
❌ ゲーム中ポーズ禁止

✅ Self-paced（無制限時間）
✅ 常時ポーズ可能
✅ 「考えてOK」というメッセージ

理由: ASD児は時間圧力でパニック・焦燥感。anxiety増加 → 学習効果低下


【禁止2】感覚オーバーロード最小化
──────────────────────────────────

ビジュアル:
  ✗ 複雑背景パターン
  ✗ 不要なアニメーション
  ✓ シンプルグラフィック（レトロゲーム風）
  ✓ 高コントラスト（視認性優先）

オーディオ:
  ✗ 高音域の突発的効果音
  ✗ 大音量BGM
  ✓ 低音量、ミュート可能BGM
  ✓ 落ち着いたトーン

情報量:
  ✗ 1画面 > 5要素
  ✗ 複雑なメニュー（3層以上）
  ✓ 1画面 ≤ 5要素
  ✓ 簡潔テキスト（< 1文）


【禁止3】予測可能性・構造化の強調
──────────────────────────────────

✅ ゲームルール不変
✅ UI配置常に同じ位置
✅ レベル進行は線形/予測可能
✅ 「次は何か」が明白

❌ Surprise rule changes
❌ 予期しないビジュアルシフト
❌ ランダムな難度変動
❌ 複雑なナビゲーション


【禁止4】エラー処理の工夫
──────────────────────────────────

❌ 大げさなエラー音
❌ スコア大幅減点
❌ 「間違い！」直接的メッセージ

✅ サイレント・エラー修正（正解への再提示）
✅ 中立的フィードバック（「もう一度試してみましょう」）
✅ プログレッシブヒント
✅ スコア据え置き（エラー記録なし）

根拠: Errorless Learning原則 → 習得速度向上、anxiety低下

5. 適応型難度調整アルゴリズム
================================================================================

【推奨】Success Rate Target = 75-80%
───────────────────────────────────────

根拠:
  • 70% < SR < 90% → 最大学習効果
  • SR ≈ 80% → engagement & motivation最高
  • SR < 70% → フラストレーション
  • SR > 90% → 退屈

実装:
  Trial-by-trial adjustment:
    IF success_rate(last 10 trials) < 75%:
      難度 -1 down
    IF success_rate(last 10 trials) > 85%:
      難度 +1 up
    ELSE:
      難度維持


【推奨】Fuzzy Logic-based Adaptation
──────────────────────────────────────

要素（weighted):
  1. 最近の成功率 (50%)
  2. 反応時間 (25%) — 遅い → 難度down
  3. エラーパターン (15%) — 特定タイプ → targeted
  4. セッション時間 (10%) — 疲労検出

利点:
  ✓ 単一メトリクスより個別化
  ✓ リアルタイム適応
  ✓ Overload回避


【実績】Pickstar Model
──────────────────────────────────────

  • DDA導入 → engagement +88.4%向上
  • Vocabulary learning for ASD
  • Player data → real-time adjustment → personalized

6. TOP3 認知強化ゲーム推奨
================================================================================

【TOP1】認知的柔軟性 × Executive Function ゲーム
──────────────────────────────────────────────

ゲーム形式: ルール切り替え型
  Level 1: 単一ルール（例：色分類）
  Level 2: ルール明示 + 切り替え（例：形→色）
  Level 3: 複数ルール並行判定

パラメータ:
  • 15分 × 2-3回/週 × 12-18週間
  • SR target: 75-80%
  • 強化: VR3（習得後）
  • エラー処理: Errorless learning

禁止事項:
  ✗ 時間制限
  ✗ 感覚overload
  ✗ 予期しないルール変更

期待成果:
  ✓ 抑制制御改善（週4以内）
  ✓ 認知的柔軟性（4-8週で顕著）
  ✓ 日常「切り替え」タスク易化


【TOP2】ワーキングメモリ訓練ゲーム
──────────────────────────────────────────────

ゲーム形式: マッチング + シーケンス記憶
  Level 1: 2-3個要素マッチング
  Level 2: 4-5個シーケンス記憶
  Level 3: 複数フィーチャー（色+形+位置）

パラメータ:
  • 20-30分 × 5-6週間（intensive）
  • SR target: 75-80%
  • 強化: CRF初期 → VR3-5
  • フィードバック: < 200ms 音+ビジュアル

難度調整: Fuzzy logic（SR + reaction time + error pattern）

期待成果:
  ✓ 言語的WM改善（4週以内）
  ✓ 視空間的WM改善（4-8週）
  ✓ 学習スキル全般向上


【TOP3】マッチング/パターン認識 + 適応型難度
──────────────────────────────────────────────

ゲーム形式: Visual Matching + Pattern Recognition
  Level 1: 2-3色/形マッチング
  Level 2: 色+形+位置パターン
  Level 3: 細かい差異認識

パラメータ:
  • 15-20分 × 3-4回/週
  • 装置: Tablet/Touch screen
  • SR target: 75-80%
  • 強化: VR2-3

グラフィック:
  ✓ レトロ風（シンプル、high contrast）
  ✓ Sensory-friendly
  ✗ 装飾最小

期待成果:
  ✓ 視覚的弁別力向上（2-4週）
  ✓ 集中力向上（engagement高い）
  ✓ マルチセンサリー情報処理改善

8. 実装チェックリスト
================================================================================

ゲーム設計フェーズ:
  □ 禁忌確認: 時間制限なし、sensory overload予防、予測可能性確保
  □ 強化スケジュール: CRF → VR設計済み（ABA原則準拠）
  □ 適応型難度: SR 75-80%自動調整可能か
  □ エラー処理: Errorless learning実装済み
  □ 即座フィードバック: < 200ms 音+ビジュアル
  □ デバイス: Mobile/Tablet対応（スケーラビリティ）

評価フェーズ:
  □ Sample size: n > 20（可能なら RCT）
  □ Effect size: Cohen's d報告
  □ Transfer測定: Trained skills外での改善
  □ Sustainability: セッション終了後の効果持続（follow-up）
