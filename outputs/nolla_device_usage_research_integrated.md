# 発達障害児のスマホ・タブレット使用と人気コンテンツ — 統合リサーチレポート

**作成日**: 2026-04-02
**背景**: ASD児の親へのインタビューで「タブレット/スマホを高頻度で使用」「主にRoblox・Minecraft」という情報を得た。これを起点に、発達障害児全般のデバイス使用実態と人気コンテンツを徹底調査。

---

## エグゼクティブサマリー


| 項目       | 主要データ                                       |
| -------- | ------------------------------------------- |
| 使用率      | タブレットが主要デバイス（40.3%）、定型発達児より高頻度              |
| 使用時間     | ASD児: 平均3.7時間/日、ADHD児: 3.3時間/日（定型児の約2倍）     |
| ゲーム時間    | ASD児: 2.1時間/日（定型児1.2時間/日）                   |
| 人気ゲーム    | Minecraft・Robloxが圧倒的。RPG（31%）、アクション（19%）が続く |
| 動画視聴     | 同じ動画の繰り返し視聴が特徴的。感覚刺激系・ゲーム実況・アニメが人気          |
| ゲーム依存リスク | ADHD児: 72.3%、ASD児: 45.5%がゲーム障害基準超過（一般児9.5%） |
| 親の最大ペイン  | 「切り替え困難」＝やめられない・止められない                      |

## 1. 使用率とデバイス選択

### デバイス別使用率（発達障害児）


| デバイス       | 主要使用率         |
| ---------- | ------------- |
| タブレット/iPad | **40.3%**（最多） |
| スマートフォン    | 26.0%         |
| テレビ        | 26.6%         |


- タブレットが圧倒的に主要デバイス。画面の大きさ、タッチ操作の直感性、携帯性のバランスが発達障害児に適している
- 知的障害児のタブレット利用率はやや低い（6.6%、総務省2012年調査）が、データが古い

**出典**: Vanderloo et al., 2025, PMC12264844 / 総務省 2012年障がい者ICT利用調査

### 定型発達児との比較

- ASD児はスクリーン使用量が定型発達児の**約2倍**
- 41.4%のASD児が自由時間の大部分をゲームに費やす（定型発達児は18%）
- ASD児のうちガイドライン適合は約45.3%（=約55%が過度な使用）

## 2. 使用時間（スクリーンタイム）

### 障害種別の1日あたりスクリーンタイム


| 障害種別          | 平均使用時間     | 過度な使用（推奨超過）の割合 |
| ------------- | ---------- | -------------- |
| **ASD**       | 3.70時間/日   | 30.0〜70.9%     |
| **ADHD**      | 3.28時間/日   | 12.9〜47.6%     |
| **脳性麻痺**      | 3.39時間/日   | —              |
| **定型発達児（参考）** | 約1.5〜2時間/日 | —              |


### ASD児のスクリーンタイム分布（名古屋大学・浜松医科大学共同研究）


| グループ        | 割合        |
| ----------- | --------- |
| 1日1時間程度     | 27.9%     |
| 1日2時間〜徐々に増加 | 19.0%     |
| 1日3時間程度継続   | 20.3%     |
| **1日4時間以上** | **32.8%** |


**出典**: PMC12264844 / 名古屋大学 2023年研究

### 年齢別の傾向


| 時期          | 特徴                                 |
| ----------- | ---------------------------------- |
| 乳幼児期（0〜4歳）  | 親がなだめるためにメディア活用。ASD遺伝的リスク児は早期から長時間 |
| 学童期（5〜12歳）  | 使用時間が大幅増加。ゲーム・YouTube中毒化リスク        |
| 思春期（13〜18歳） | 依存リスク最大。男子・ADHD併発・親の監督不在で悪化        |

## 3. 人気ゲーム・アプリ

### Minecraft・Robloxが圧倒的に人気な理由

#### Minecraft


| 要因                 | 詳細                           |
| ------------------ | ---------------------------- |
| **感覚カスタマイズ**       | 明るさ・音声・視覚効果を自分で調整可能。感覚過敏への対応 |
| **予測可能なルール**       | 物理法則が一定。予測可能性への強い好みを満たす      |
| **構造と自由のバランス**     | サンドボックスで自由度が高いが、ルールは明確       |
| **対面不要のコミュニケーション** | テキストチャットで社会的圧力を軽減            |
| **即時フィードバック**      | 行動→結果が明確。ADHD児の報酬系を刺激        |


**出典**: LearningWorks for Kids / moveupaba.com

#### Roblox


| 要因             | 詳細                            |
| -------------- | ----------------------------- |
| **安全な社会的練習**   | アイコンタクト・ボディランゲージ不要で社会的相互作用が可能 |
| **ワールド制作の自主性** | 自分だけの世界を作れる。コントロール感           |
| **不確実性の低さ**    | 予測可能な環境。自分のペースで参加可能           |


**出典**: Internet Matters / askellie.co.uk

### その他の人気ゲーム・アプリ


| カテゴリ    | タイトル       | 特徴               |
| ------- | ---------- | ---------------- |
| パズル系    | 各種パズルゲーム   | ルール明確、達成感        |
| LEGO系   | LEGO games | 構造化された創造遊び       |
| プログラミング | Scratch    | ブロックプログラミング、論理思考 |


### ASD児のゲーム内行動パターン（定型児との違い）

- **一つのモードに深く没頭**: 同じマップ・同じプレイを繰り返す
- **ルールの厳格な遵守**: ゲームルールからの逸脱を嫌う
- **ソロプレイ傾向**: マルチプレイより単独プレイを好む
- **RPGへの高い親和性**: 31%が最も好むジャンル（明確な進行・達成感）

## 4. 人気の動画コンテンツ

### ASD児のYouTube視聴パターンの特徴

- **同じ動画の繰り返し視聴**が最大の特徴
  - 理由: 予測可能性による安心感 / 感覚の自己調整 / 毎回の新しい発見 / 言語習得
- **視覚学習者が多い**ため、動画との親和性が非常に高い
- COVID-19パンデミック時調査: YouTube利用が80%以上で増加

### 人気ジャンル


| ジャンル            | 人気度   | 具体例                                         |
| --------------- | ----- | ------------------------------------------- |
| **感覚刺激系（ASMR）** | 非常に高い | スライム動画（ビーズ・スパンコール混合）、タッピング音                 |
| **ゲーム実況**       | 非常に高い | Minecraft実況、RPGプレイ動画                        |
| **アニメ**         | 高い    | ドラえもん、クレヨンしんちゃん、ポケモン、SPY×FAMILY、僕のヒーローアカデミア |
| **教育系**         | 中〜高い  | Super Simple Songs、Ms. Rachel（幼児向け）         |


### 日本で人気のコンテンツ


| プラットフォーム         | 人気コンテンツ                                     |
| ---------------- | ------------------------------------------- |
| **NHK Eテレ**      | 「いないいないばあっ！」（0〜2歳最強）、「おかあさんといっしょ」、「えいごであそぼ」 |
| **YouTube**      | あっちゃんファミリー（1,470万人）、ゲーム実況、スライムASMR          |
| **Netflix**      | Curious George、ポケモン、各種アニメ                   |
| **Amazon Prime** | アニメ全般                                       |


### 繰り返し視聴の科学的背景


| 機能   | メカニズム             |
| ---- | ----------------- |
| 言語習得 | キャラクターの台詞から繰り返し学習 |
| 感覚調整 | 聴覚・視覚の自己統制ツール     |
| 不安軽減 | 予測可能な世界への没入       |
| 認知発達 | 毎回異なる視点での理解深化     |


**出典**: 早稲田大学研究 / JAMA Pediatrics / PMC各論文

## 5. なぜゲームにハマりやすいのか — 神経学的メカニズム

### ASD児の場合


| 要因            | 説明                              |
| ------------- | ------------------------------- |
| **制限的興味との適合** | 特定のゲーム世界・ルール体系が「限定された興味」を完璧に満たす |
| **予測可能性への嗜好** | 脳の認知負荷が軽減され、超集中を維持しやすい          |
| **感覚カスタマイズ**  | ゲーム環境の感覚設定調整で過負荷を回避 → 没頭可能      |


### ADHD児の場合


| 要因            | 説明                                |
| ------------- | --------------------------------- |
| **即時報酬**      | スコア・レベルアップ等の即座フィードバックがドーパミン系を強く刺激 |
| **ハイパーフォーカス** | 興味あるタスクへの異常な集中力。報酬依存性が強い          |
| **実行機能の逆転**   | 学業では発揮できない集中力がゲーム環境では最大化          |


### 共通メカニズム

- ゲームの「即座的・構造化された」環境は、従来型教育より神経多様性のある脳に適合
- 「ゲーム依存」は単なる意志力の問題ではなく、**脳の報酬系・実行機能の特性**に起因

**出典**: PMC7851038 / Frontiers in Pediatrics 2025 / PMC9340127

## 6. ゲーム依存・スクリーン依存のリスクデータ

### 障害種別のゲーム障害率


| 対象        | ゲーム障害基準超過率            |
| --------- | --------------------- |
| 一般児       | 9.5%                  |
| **ASD児**  | **45.5%**             |
| **ADHD児** | **72.3%**             |
| ASD+ADHD  | インターネット依存 20〜22.2/100 |


### リスク要因


| 要因        | 相対リスク           |
| --------- | --------------- |
| 男性        | 3.7倍            |
| ADHD症状共存  | 高リスク            |
| 親の監視ルール欠如 | 高リスク            |
| 母子家庭      | ゲーム障害児の43%      |
| 睡眠障害      | ゲーム障害児の59%が入眠障害 |
| 不登校       | 36%がネット依存リスク    |


### 国内統計

- ネット・ゲーム依存疑いの中高生: **93万人**（中高生の7人に1人）
- 高校生の約10%に依存疑い
- ADHD児は4時間超使用で、3時間未満児と比べADHD症状報告が**7.5倍**

**出典**: 厚労省2017推計 / PMC10887068 / ScienceDirect

## 7. 医学的ガイドラインの現状


| 機関               | 推奨                    | 発達障害児への特記            |
| ---------------- | --------------------- | -------------------- |
| **WHO**          | 5歳以上: 2時間以下/日         | 発達障害児には適用困難との指摘あり    |
| **AAP（2026年更新）** | 時間制限→「質・文脈・親子対話」重視に転換 | 補助技術としてのスクリーン有用性を認める |
| **日本小児科学会**      | 独自ガイドラインあり            | 発達障害児への特別言及は限定的      |


**重要な因果関係の整理**:

- 「スクリーン使用が長い → 発達障害になる」ではない
- 「発達障害の特性 → スクリーン使用が長くなる」が正確
- 長時間使用は発達障害の**早期特性の表れ**である可能性（大阪大学2023年研究）

## 8. 親の困りごと（ペインポイント）

### 困りごとランキング


| 順位  | 困りごと                      | 深刻度 |
| --- | ------------------------- | --- |
| 1   | **切り替え困難**（やめられない・止められない） | 最高  |
| 2   | **スクリーン依存の進行**            | 高   |
| 3   | **睡眠障害**（就寝前の使用→入眠困難）     | 高   |
| 4   | **行動悪化**（使用後の落ち着きのなさ・衝動性） | 中   |
| 5   | **社会性低下**（リアルの遊び・人間関係の後退） | 中   |


### 発達段階別の課題


| 時期     | 親の主な悩み                   |
| ------ | ------------------------ |
| 0〜3歳   | 親自身がなだめるためにスマホに頼ってしまう罪悪感 |
| 3〜12歳  | ゲーム・YouTubeを止める時の癇癪・パニック |
| 13〜18歳 | SNS依存・ゲーム障害化・不登校への発展     |


**出典**: 東京都福祉保健局 / PRESIDENT Online / AIQ School

## 10. Nollaへの示唆

### インタビュー結果の裏付け

「タブレット/スマホを高頻度で使い、Roblox・Minecraftをやる」というインタビュー結果は、研究データと**完全に一致**する。これは一家庭の特殊事例ではなく、発達障害児全般に共通する行動パターン。

### プロダクト設計への示唆

1. **「すでにデバイスを持っている」が前提で設計できる** — タブレット/スマホ向けアプリとしての展開が妥当（普及率40%超・使用時間3.7時間/日）
2. **「ゲームの仕組み」を療育に転用する余地が大きい**
  - 即時フィードバック・予測可能なルール・感覚カスタマイズ — これらがMinecraft/Roblox人気の本質
  - 同じ原理を療育コンテンツに組み込めば、「やめられない」を「自然に支援を受けている」に転換できる可能性
3. **親の最大ペイン「切り替え困難」への直接的アプローチ**
  - 依存的な使用（ゲーム4時間+）→ 建設的な使用への誘導
  - ビジュアルタイマー・段階的終了予告などのUX設計
4. **「動画の繰り返し視聴」は学習メカニズムとして活用可能**
  - 繰り返し視聴は「病的行動」ではなく自己調整・学習の手段
  - 療育コンテンツをこのパターンに乗せれば高いエンゲージメントが期待できる
5. **依存リスクへの責任ある設計**
  - ASD児45.5%・ADHD児72.3%がゲーム障害基準超過という現実
  - 「使わせすぎない」仕組みを組み込むことで、親の信頼を獲得できる差別化要因になる
