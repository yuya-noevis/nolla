# Nolla — サービス概要

---

## エグゼクティブサマリー

**Nollaは、認知機能に課題のある子ども（3-18歳）の発達を、AIで個別最適化して支援するプラットフォームである。**

家庭・施設・学校の3つの場所にまたがる支援の分断を、一つの共通基盤で接続する。子どもにはパーソナライズされた「あそび」を、親にはAIコーチングを、施設・学校にはリアルタイムの発達ダッシュボードを提供する。

**対象市場：** グローバル約4億人（年間約10%増加）、日本国内約300万人
**直接競合：** 日本市場でゼロ。グローバルでも3障害横断×3場所統合のプロダクトは存在しない

---

## 課題

### 前提：早期介入の効果は科学的に実証済み

早期に効果的な介入があれば、知能・社会性など脳の機能は向上し、子どもの可能性が拡大する。家族の負担も軽減される。

### にもかかわらず、効果的な介入が届いていない

| # | 課題 | 現状 |
|---|---|---|
| ① | **能力向上の手段がない** | 認知機能を向上させるスケーラブルなアプローチが存在しない |
| ② | **行動変容の手段が不足** | ABA（応用行動分析）は高コスト・高人材依存。すべての対象者に届かない |
| ③ | **家庭に届かない** | 専門的介入は週2時間。残り週166時間は支援の空白 |
| ④ | **施設・学校の質に格差** | 支援の質が担当者の経験に依存。効果測定の手段もない |
| ⑤ | **人力では個別最適化できない** | 子どもの特性は多次元。一人ひとりに最適な支援を人力で設計・実行することは構造的に不可能 |
| ⑥ | **コスト増加が止まらない** | 子どもの能力が向上しない限り、ケア負担とコストは構造的に増加し続ける |

### 課題の本質

**科学的に効果が実証された介入は存在する。しかし、それを個別最適化し、家庭に届け、継続させる手段がない。** これは医療の問題ではなく、デリバリーとパーソナライゼーションの問題である。

## ソリューション

### ワンライナー

**Nolla =「うちの子の"できた"が見える」— 家庭・施設・学校をつなぐ、AI個別最適化の発達支援プラットフォーム**

### プロダクトの本質

Nollaは3つの異なる場所に、3つの異なる価値を提供する。しかし裏側では**一つの共通基盤**が動いている。

## 3つのビジネスモデル

### A. 家庭（ToC）— 成長・療育

**ユーザー：** 子ども（6-12歳）＋保護者

#### 子どもの体験

毎日15-20分の「たのしいあそび」。子どもにとってはゲーム。裏側ではエビデンスベースの認知＋身体複合介入が動いている。

| 要素 | 内容 | 科学的根拠 |
|---|---|---|
| **認知＋身体ゲーム7種** | 抑制制御、WM、認知柔軟性、計画、注意切替。うち最低2種は身体動作を含む | 身体活動×認知の複合介入で実行機能 g=0.40-0.83（Frontiers 2025） |
| **音楽プライミング** | セッション冒頭3分。リズム同調で覚醒状態を最適化 | 音楽療法のプライミング効果。覚醒調整が共同注意のメディエーター |
| **適応型難易度調整** | 正答率70-80%を維持するDDA。フロー状態を持続 | MLベースDDA確立済み（ScienceDirect 2024）。EndeavorRxの閉ループ実績 |
| **非線形ペダゴジー** | 同じスキルを毎回異なる文脈で練習。汎化促進 | 線形的指導より習得・保持で優位（Frontiers in Psychology 2025） |
| **障害タイプ別報酬** | ASD=予測可能な完了感、ADHD=新奇性と即時FB、ID=温かい成功の蓄積 | 障害特性ごとの報酬系の違いに基づく設計 |
| **ナビゲーターキャラクター** | 伴走するが依存させない。自発的な感情連絡は禁止 | ロボット介入の知見（予測可能性＋社会的単純さ） |

#### 保護者の体験

| 機能 | 内容 |
|---|---|
| **成長きろく** | 5軸認知プロファイルの経時変化。「昨日より注意が12秒伸びた」をLLMが自然言語で翻訳 |
| **今日のミッション** | 子どものデータに基づき、家庭でできる具体的行動を毎日提案。「夕食準備で3つの食材から選ばせてみて」 |
| **AIコーチ（質問チャット）** | 「パニック時どうすれば？」→ 子どものプロファイル（感覚特性、過去の対処効果）に基づく個別化回答 |
| **SOSモード** | メルトダウンの最中に開くと、3行以内の即時対処ガイドを表示 |
| **共有ダッシュボード** | 共有範囲を保護者が設定。施設や学校の担当者もリアルタイムで閲覧可能 |

#### このモデルが解く課題

- **③家庭の空白：** 週166時間の空白を、毎日15分のゲーム＋親AIコーチで介入機会に変換
- **①能力向上：** 認知＋身体複合介入（g=0.40-0.83）で実行機能を向上
- **②行動変容：** LLMがABA/PBSの原則を親に翻訳。覚醒調整で問題行動を予防
- **⑥コスト：** 親の介入スキル向上→困りごと減少→ケア負担軽減

### B. 施設（ToB）— 成長・療育支援

**ユーザー：** 放課後等デイサービス・児童発達支援事業所の支援スタッフ＋施設長

**市場：** 国内約2万施設（放デイ＋児発）、市場規模4,669億円（2022年度）。4割が赤字。

#### 支援スタッフの体験

| 機能 | 内容 |
|---|---|
| **リアルタイムダッシュボード** | 担当児童の認知プロファイル、家庭でのゲームデータ、直近の状態変化が一目でわかる |
| **今日の支援ガイド** | AIが児童の最新データから「今日推奨されるアプローチ」を提示。「昨日は睡眠不足。低負荷の活動から始めて」 |
| **セッション記録** | 施設での観察をタップ入力 → 家庭データと自動統合 |
| **施設内ゲームセッション** | 家庭版と同じゲームコンテンツを施設の療育時間に活用 |

#### 施設長の体験

| 機能 | 内容 |
|---|---|
| **全児童の進捗一覧** | 誰が伸びていて、誰に追加支援が必要か |
| **効果の可視化** | 「この支援を受けた児童のX%がY領域で改善」が自動で可視化される |
| **スタッフ別効果** | どのスタッフの支援が成果を出しているか。質の格差を数値で把握 |

#### 施設が得る価値

- **効果測定の自動化：** 「この子に何が効いているか」が初めて客観的にわかる
- **属人性の排除：** 経験の浅いスタッフでもAIガイドに沿って質の高い支援を提供可能
- **家庭との連携：** 施設の2時間と家庭の166時間が初めてデータでつながる
- **保護者との信頼構築：** 共有ダッシュボードで保護者がリアルタイムに進捗を確認。「何をやっているか見えない」問題を解消

#### このモデルが解く課題

- **④施設の質：** 効果測定の自動化＋AIガイドで、支援者の経験差に依存しない品質を実現
- **⑤個別最適化：** 家庭＋施設のデータ統合で、単一場所では見えない子どもの特性を把握

### C. 学校（ToB/ToG）— 教材として学習支援

**ユーザー：** 特別支援学級・通級の教師＋教育委員会

**市場：** 特別支援教育の対象児童生徒数は年々増加。2023年度は通常学級の8.8%に発達障害の可能性。

#### 教師の体験

| 機能 | 内容 |
|---|---|
| **個別最適化教材** | 認知プロファイルに基づき教材を自動調整。WMが弱い子には指示を1ステップずつ提示、視覚優位の子にはテキストを図解に変換 |
| **IEP作成支援** | 認知プロファイルから「この児童の強みと課題」を自動要約。目標設定の提案。進捗の自動追跡 |
| **リアルタイムダッシュボード** | 保護者が共有を許可した範囲で、家庭・施設でのデータを閲覧。児童の全体像を把握 |

**表面の言葉は「教育」の文脈に完全に切り替える：**
- 認知ゲーム → 「集中力トレーニング」「考える力あそび」
- 実行機能課題 → 「計画を立てる練習」「切り替えチャレンジ」

#### 教育委員会の体験

- 管轄校の特別支援教育の効果を横断的にモニタリング
- エビデンスに基づく予算配分の判断材料

#### 学校が得る価値

- **教材作成の工数削減：** 個別最適化を人力でやる負担からの解放
- **IEPの客観化と効率化：** データに基づく計画策定と評価
- **3場所のデータ共有：** 家庭・施設の状態を把握した上での指導が可能に（保護者同意前提）

#### このモデルが解く課題

- **④学校の質：** 個別最適化教材で教師の経験差に依存しない学習支援
- **⑤個別最適化：** 認知プロファイル＋DDAで一人ひとりに合った教材を提供
- **①能力向上：** 学校の個別学習時間に認知課題を日常的に提供

## 3つの場所をつなぐ共有ダッシュボード

Nollaの構造的差別化は、家庭・施設・学校のデータが**一つのダッシュボード上でリアルタイムに共有される**こと。

### 仕組み

- **保護者が共有範囲を設定する。** 何を、誰に、どこまで共有するかは保護者が完全にコントロール
- 共有を許可された施設・学校の担当者は、同じダッシュボードをリアルタイムで閲覧
- レポート生成は不要。ダッシュボード自体が「今の状態」を常に示している

### これが生む価値

| 従来 | Nolla |
|---|---|
| 施設での様子を親が知るのは送迎時の口頭報告のみ | ダッシュボードでリアルタイムに把握 |
| 学校の先生は家庭の状況を知らない | 共有範囲内のデータを即時閲覧 |
| 施設と学校の情報共有がゼロ | 保護者を介して間接的に接続 |
| 「何が効いているか」を誰も把握していない | データの横断分析で効果を可視化 |

### データ統合の原則

- 生データはデバイスから出ない（エッジAI処理）
- 共有されるのは匿名化された特徴量と進捗サマリーのみ
- 保護者がいつでも共有を停止可能
- COPPA 2025完全準拠

## コア技術

### 1. 認知プロファイルエンジン

診断名ではなく、5軸＋感覚＋覚醒特性で子どもを多次元的に理解する。

| 軸 | 測定内容 |
|---|---|
| 実行機能 | 抑制制御、認知柔軟性、計画 |
| 注意 | 持続的注意、選択的注意、注意切替 |
| ワーキングメモリ | 視空間WM、言語WM |
| 感覚処理 | 感覚過敏/鈍麻の傾向 |
| 社会認知 | 感情認識、意図理解（V2以降） |
| 覚醒特性 | 覚醒レベルの変動パターン |

ゲームプレイ自体がアセスメント。テストと介入の境界がなくなる。

**根拠：** 実行機能の遅れはASD・ADHD・LD・ID・境界知能を横断する「共通課題」（Nature Human Behaviour 2024）。診断名ではなく認知プロファイルでのアプローチが科学的に合理的。

### 2. 3層AIエンジン

| 層 | 手法 | 時間軸 | 最適化対象 |
|---|---|---|---|
| **リアルタイム** | Multi-Armed Bandit | 試行（数秒） | 刺激の難易度・種類・フィードバック |
| **戦略** | Bayesian Optimization | セッション〜週 | 介入の組み合わせ・強度・順序 |
| **予測** | 発達軌跡モデリング | 月〜年 | 変化の気づき・停滞の早期検知 |

リアルタイムAIは短期エンゲージメントに、戦略AIは長期学習成果に、予測AIは発達軌跡の理解に、それぞれ最適化される。

**MVPではバンディット＋ルールベースから開始し、データ蓄積に伴いフルRLエンジンへ段階的に移行。**

### 3. LLMロール別翻訳

同じ子どものデータを、ユーザーの役割に応じて異なる文脈・言葉に変換する。

| ロール | 翻訳例 |
|---|---|
| **親** | 「今日は集中が15%伸びました。夕食準備で3つの食材から選ばせてみてください」 |
| **施設スタッフ** | 「昨日の睡眠が短い。今日は低負荷の感覚遊びから開始を推奨」 |
| **教師** | 「WMが弱いため、指示は1ステップずつ提示。視覚的な補助が有効」 |

一つのエンジンが3つの市場に価値を届ける。

### 4. N-of-1エビデンス基盤

各子どもが「自分自身の対照群」として機能。異なる介入内容のA/Bテストにより「何がこの子に効くか」を個別に特定する。

- 非介入期は設けない（倫理的配慮）
- 5,000人のN-of-1データが集まれば、RCTに匹敵するエビデンスに
- Day 1からデータスキーマに組み込む

### 5. プライバシー基盤

- 全ての生データ（音声・映像・生体信号）はオンデバイスで処理
- クラウドには匿名化済み特徴量のみ送信
- 連合学習で個人データを共有せずに集団パターンを学習
- COPPA 2025完全準拠（2026年4月期限）

## エビデンス基盤

### 活用する介入アプローチとエビデンス強度

| ランク | 介入 | 効果量 | テクノロジー代替可能性 |
|---|---|---|---|
| **S** | 早期集中行動介入（EIBI/ESDM） | IQ +17.6pt | AIで判断部分を効率化 |
| **A** | 親実施型介入（PMI） | g = 0.553 | **AIコーチングで拡張可能** |
| **A** | 有酸素運動×認知の複合介入 | 抑制制御 d=0.83, WM d=0.48 | ゲーミフィケーション可能 |
| **A** | 社会スキル訓練（SST） | g = 0.81-0.93 | **AI/VRで代替可能** |
| **B+** | マルチドメイン認知トレーニング | 単一ドメインより優位 | **極めて高** |
| **B+** | デジタルセラピューティクス | FDA承認済み（EndeavorRx） | **極めて高** |

### 効果的な介入の共通原則（Nollaの設計に反映済み）

1. 早期であるほど効果が大きい
2. 個別化が効果を最大化する
3. マルチドメイン（複数領域の同時介入）が単一より効果的
4. 親の関与が効果の持続と般化を高める
5. 低強度でもターゲットを絞れば効果的（ESDM週3時間でも有効）
6. 継続が命（中止後は効果減衰）
7. 生活習慣（睡眠・運動）が神経可塑性を増幅

### 主張可能な効果と主張不可の境界

| コンポーネント | 主張可能 | 主張不可 |
|---|---|---|
| 認知ゲーム | 訓練した認知スキルの向上（近転移） | IQ向上、学業改善（遠転移） |
| AIコーチ（PMI） | 親のスキル向上、関わりの質の改善 | 専門家の完全代替 |
| 運動チャレンジ | 実行機能・抑制制御の向上 | 運動能力の劇的改善 |
| デジタルフェノタイプ | 認知プロファイルの継続的モニタリング | 医学的診断 |

### 防御的モート

1. **データモート：** 数万人の縦断的発達データ。時間が経つほど蓄積し、後発は追いつけない
2. **アルゴリズムモート：** データに基づく介入最適化AI。データなしには再現不可能
3. **ネットワーク効果：** 施設→家庭→学校のエコシステム。接続が増えるほどデータの価値が上がる

## なぜ今か

1. **AI技術の成熟：** LLM（Claude/GPT-4）、オンデバイスAI（ExecuTorch GA）、デジタルフェノタイピング（AUC 0.92）が実用段階に到達
2. **規制の前例：** ENDEAVORRIDEの国内承認（2025年）がDTxの道筋を開いた
3. **社会的認知の変化：** 発達障害の診断数が急増。支援需要は爆発的に拡大している
4. **競合の空白：** 日本市場に直接競合ゼロ。グローバルでも3障害横断×3場所統合は存在しない
5. **コスト構造の限界：** ABAの人的モデルでは需要に追いつけない。テクノロジーによる代替は不可避
